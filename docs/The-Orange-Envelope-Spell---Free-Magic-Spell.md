# 橙色信封咒语

一个可用于在需要额外好运时一次又一次使用的好运咒语。

## '橙色信封咒语' 施法说明

**你将需要以下物品来施展这个咒语：**

+   1 绿色叶子 [任意]

+   1 黄色花朵 [任意]

+   1 蓝色花朵 [任意]

+   一个碗

+   一把泥土

+   1 橙色信封

**你将需要以下物品来施展这个咒语：**

+   1 绿色叶子 [任意]

+   1 黄色花朵 [任意]

+   1 蓝色花 [任意]

+   一个碗

+   一把泥土

+   1 橙色信封

~ 握住叶子在你的力量手中：

绿色；大自然的颜色，我摘下了这片叶子

由于这片叶子是绿色的，请给我带来好运。

无害他人，如此成就吧。

~ 用叶子仍在你的力量手中，用同一只手拿起花朵：

黄色花在大地上生长，

请在我身上赐予好运。

无害他人，如此成就吧。

~ 现在，仍然握着叶子和黄色花的情况下：

你的命令，精神的颜色

我请求通过这个咒语得到祝福。

无害他人，如此成就吧。

~ 将三者放入碗中，并在撒土时说：

当我撒上母亲的土壤时

愿我记得我仍然

必须努力工作，我必须辛勤劳作

如果我要实现我的意愿。

无害他人，如此成就吧。

~ 集中精力在碗上，并重复最后的祈祷两次，看到自己获得好运。一旦你感觉它完全充满能量，将内容倒入橙色信封中，并将其放在不会被打扰的地方。每当你觉得需要更多好运时，拿出来，双手握住它[就像在祈祷]，专注于好运帮助你。一定要感谢神灵为你从这个咒语中获得的好运。
