# 龙族净化咒语。

这个净化咒语（或用龙语说：''Koeans'lin chedayret'’）有助于清除你的身体上的负面影响、诅咒、魔咒和其他不良影响。

## 施法指南 '龙族净化咒语'

**你将需要以下物品来施展这个咒语：**

+   Sage（注：叶子必须像薰香一样干燥并捆扎在一起）

+   龙血香

+   并且一块净化/保护石，如：玉、黑曜石、缟玛瑙或日光石。
