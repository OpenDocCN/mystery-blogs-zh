# 失恋接受法术 - 学习巫术

> 来源：[https://learningwitchcraft.com/acceptance-spell-for-a-love-gone-astray/#0001-01-01](https://learningwitchcraft.com/acceptance-spell-for-a-love-gone-astray/#0001-01-01)

不幸的是，爱情并非在所有情况下都是永恒的。有时，它可能会变成冷漠。如果你的前任需要帮助接受事情已经改变，这个法术可以帮助。不要在星期二进行。

收集以下物品。找到南瓜籽和铲子。画一个你前爱情朝着新对象咧嘴笑的图片。

带上南瓜籽走到户外找一个可以独处的地方。使用铲子挖一个小洞。通过鼻子吸气，然后屏住一分钟。从嘴里呼出。重复几次。将你的前任想象成就在你身边。

现在，想象你的旧爱在走远。当他或她朝着地平线前进时，咬一颗南瓜籽，然后将其放入洞中。一直做到你的前任完全从画面中消失为止。

想象一下没有这段关系后你的新生活会是什么样子。想象所有与这种改变相关的积极因素。思考这次分手对你们两个来说会有多好。当你把最后一个南瓜籽放进洞里后，再加上你前任的画像。把所有的土覆盖在上面，一边心怀善意地为你前爱情的新生活祝福。一旦完全覆盖，站起来，转身离开，保持目光前视（不回头）。带着幸福和接受进入你的新生活。
