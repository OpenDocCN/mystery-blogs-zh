# 梦中问题，梦中答案

这个咒语将让你在梦中找到许多问题的答案。

## 施法说明：“梦中问题，梦中答案”

+   1支白色蜡烛（可选）

+   1支肉桂香薰棒

+   一张纸条

+   一支钢笔

亲爱的阿芙洛狄忒，

“这是我寻求的答案，
