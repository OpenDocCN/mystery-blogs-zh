这是我为了帮助我未来的试镜而设计的咒语。请记住，每个咒语都有意图和行动。如果您不采取行动，不要指望咒语能起作用。在一切中都要放入您的直觉。最重要的是，要进行研究。这个咒语适用于戏剧、乐队试镜等。随意根据您的喜好更改措辞。祝福。

**进行此咒语，您将需要以下物品：**

+   9片月桂叶

+   9块白檀木，最好是大块

+   薄荷、鼠尾草和柳树

+   两支蜡烛：一支紫色，另一支代表您希望试镜的音乐剧或角色的颜色

+   盐

+   一个碗

+   熏香。任何都可以，但最好是白檀木，用于许愿，或者龙血香，增加咒语的效力。

+   可选 - 您的直觉对这个咒语被吸引的任何水晶。我被引导选择透明石英和玫瑰石英。这是可选的。

## 《一个愿望咒语，用于实现音乐剧角色的选拔》的施法说明

*   **如果可以的话，用您的熏香净化一切，如果选择，请绕圈，呼唤任何神灵或神圣存在来帮助您完成这个咒语。**

+   **用盐填充碗。建议使用宽口碗。**

+   **接下来，用以下文字刻上您的紫色蜡烛：**

    实现

    天赋

    愿望

    您可以选择这些词中的一个，或者全部选择。 **

+   **现在，用您选择的颜色在下面的东西上写上刻印：

    您的全名

    您想要或正在试镜的音乐剧/角色**

+   **将这些蜡烛放入盐碗中，或者如果您感觉更舒适，放在碗的两侧。您也可以使用薄荷精油涂抹草药，以帮助其粘附在蜡烛上。**

+   **现在，在所有9片月桂叶上写下一个符合您想要/正在试镜内容的肯定语句。**

+   **肯定语句是以已经发生的方式说出的陈述，所以像**

+   **“我是”**

+   **“我有”**

+   **“我将” ** **都是开始肯定语句的好方式。不要使用“我不能”、“我不会”、“我不会”的任何表述。这样很可能不会起作用。我推荐的肯定语句是“我已经被选为（角色名）来自（音乐剧名）”或“我非常适合扮演（角色名）来自（音乐剧名）”。**

+   **与白檀木碎片做同样的事情。如果您有像我一样的小块，可以跳过此步骤。 **

+   **将9片月桂叶放入盐中，呈顺时针方向排列，以吸引您的显现。**

+   **将白檀木碎片与其他草药一起放入。**

+   **随意想出任何咒语，符咒等，但这并非必需。**

+   **点燃两根蜡烛。确保你心中怀着意图。想象一下你被施咒后的感觉。**

+   **让蜡烛自己燃烧到尽头。如果由于危险原因你必须熄灭火焰，比如，你要睡觉了，确保你试着扑灭蜡烛，这样咒语就“”暂停””了，然后你可以在准备好的时候重新点燃它们。不要将这些蜡烛或草药再用于其他物品。
