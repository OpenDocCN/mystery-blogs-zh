# 看看他/她是否爱你

这是一个用来查找你的爱人是否爱你的咒语。

## 施法指南 '看看他/她是否爱你'

**你将需要以下物品来施展此咒语：**

+   一张纸（尺寸与笔记本一样大）。

+   粉红色蜡笔

+   银色/金色闪光

+   粉红色闪光（大部分需要）

+   胶水

+   更大的盘子

+   糖

**你将需要以下物品来施展此咒语：**

+   一张纸（尺寸与笔记本一样大）。

+   粉红色蜡笔

+   银色/金色闪光

+   粉红色闪光（大部分需要）

+   胶水

+   更大的盘子

+   糖

你所要做的就是用一张纸切出手掌大小的心形。做两个心形。然后你需要把它们涂成粉色。现在从侧面涂上胶水，撒上粉红色闪光。在心形的中间写上你和你心爱的人的名字（每个心形一个名字）。完成后，将它们放在水中。确保它们在水中完整。然后在水中加入糖。第二天早上检查心形。如果你的心形更受损，那说明你对他的爱不够深。如果他/她的心形受损更多，那说明他/她对你的爱不够深。你在晚上做这个。第二天早上你要做的第一件事就是把心形拿出水外。
