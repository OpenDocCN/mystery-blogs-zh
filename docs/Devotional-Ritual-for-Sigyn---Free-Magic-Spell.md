一场致力于希金的虔诚仪式。这来自于拉文·K和加琳娜·克拉斯科瓦的北欧传统独行者。

## '希金的虔诚仪式'施法说明

**你将需要以下物品来施展这个咒语：**

+   一个大碗

+   祭坛空间

+   食物（自选）

+   饮料（自选）

+   2支蜡烛

+   熏香

**你将需要以下物品来施展这个咒语：**

+   一个大碗

+   食物（自选）

+   熏香

开始设置一个专门献给希金的祭坛。将一个大碗放在祭坛中央。在这碗的左侧放一支蜡烛，在右侧放另一支蜡烛。在诵读以下祷告时点燃这两支蜡烛：

“希金，洛基的挚爱妻子，纳尔维和瓦利的母亲，温柔女士，坚韧女士。我向你致敬。赐予我你的勇气，那是一颗开放但坚不可摧的心。赐予我你的谦逊，那是对你爱的虔诚心灵的关注。最重要的是，请赐予我为你、你的丈夫、你的孩子和所有可能需要我的众神服务的能力。万岁，希金。”

取一部分食物和饮料（最好是你打算吃的一顿饭的一部分），并将其放在祭坛上供奉给希金，同时说：“为了你在洞穴中的时间，当你饥饿和营养不良以履行爱的义务。愿它滋养你。”双手拿着碗，将其举过头顶，“愿我代你拿着这碗，女士。愿我今晚在洞穴中代替你，仅仅片刻。”将碗举过头顶一段确定的时间（尽你感觉对的时间），象征着希金为洛基所做的事情。

结束时，熄灭蜡烛并点燃一根香。让香燃烧完毕作为一种供奉。将食物留在祭坛上直到第二天早晨。
