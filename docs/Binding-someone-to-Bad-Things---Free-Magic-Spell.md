# 将某人绑定到坏事上

评分为5/5星

如果你可以把一个人绑定到另一个人身上，为什么不能把他们绑定到坏事上，比如疾病、事故、金钱问题。

## “将某人绑定到坏事”的施法说明

**你将需要以下物品来施展这个咒语：**

+   黑色蜡烛

+   黑色绳/绳索（你必须有这个）

+   人物照片/纸上的名字

**你将需要以下物品来施展这个咒语：**

+   黑色蜡烛

+   黑色绳/绳索（你必须有这个）

+   人物照片/纸上的名字

如果你想将某人绑定到像疾病这样的单一事物，那很好，但如果你想同时将他们绑定到不同的事物上，拿起绳子，说出你想要的事物并打个结，继续下去。比如“死亡”打个结，“事故”打个结，“搞砸的关系”打个结。

好了，深吸一口气。点燃蜡烛。想象那个“事故”中的人，看到他们哭泣，流血等等，或者如果你不想做这一切，只看到他们悲伤、哭泣、抑郁，说“我的生活是地狱”或其他什么。玩得开心点。保持一段时间，然后感受到喜悦充满你的灵魂，因为它已经发生了。拿起绳子，捆绑图片/名字并说“我将你______束缚，我将你束缚在（事故、疾病、财务问题）之中”，做完后，我会把绑定烧成灰烬并埋葬它们，但如果你不能埋藏它......要埋得很深！

* 你可以召唤灵魂，如果你愿意，但我个人不会这样做，因为我会让它们毁灭，而不是我和一个灵魂。

* 如果你觉得更合理，你也可以拿一张纸，写下你想要将这个人绑定的坏事，或者找到那件事的符号，然后将他们绑定在一起。
