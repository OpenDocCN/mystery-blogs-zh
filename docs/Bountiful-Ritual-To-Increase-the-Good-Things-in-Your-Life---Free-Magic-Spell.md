**增加生活中美好事物的丰盛仪式**

这是一个在感恩节期间进行的美好仪式，但任何时候你想感恩你所拥有的并确保好事如金钱、爱情、成功和健康继续降临到你身上的时候都可以进行。

**你需要以下物品来施展这个咒语：**

+   粉晶、无花硝、玛瑙、石榴石、黄水晶、红玉髓、翡翠、烟晶石、方钠石和护身符袋

+   丰盛宝石袋：

+   粉晶为友谊和爱情

+   无花硝为健康的头脑

+   玛瑙为强大的保护

+   石榴石为好运、爱情和财富

+   黄水晶为成功和好运

+   红玉髓为健康和财富

+   翡翠为爱情和健康

+   烟晶石为能量和财富

+   方钠石为和谐

+   3个带明亮玻璃座的小蜡烛

+   1个白色小蜡烛、1个橙色小蜡烛和1个绿色小蜡烛

+   装有水和常青植物的花瓶

+   可以是常春藤、冬青或松树

+   1分钱，1角钱和1美分

+   一个小碟或小碗

+   银、黄铜或透明玻璃

+   1茶匙蜂蜜和一小块面包放在白色盘子上

+   一杯清水或葡萄汁的红酒

+   你感恩的10个或更多事物的清单

+   清洁你的宝石：

+   用你喜欢的方法清洁你的宝石。

+   仪式前一晚写下你的感恩清单，把常青植物放在花瓶里。

## 用于“增加生活中美好事物的丰盛仪式”的施法说明

11月是心灵感应思维强烈的时候。在11月进行这个仪式将带来爱情、财富和健康的丰盈。

为了获得爱情、财富和健康的丰盈，在满月前的周六或周日进行这个仪式。这个仪式一年不要超过3次。

准备你的祭坛：

把所有物品放在你干净的祭坛上。把蜡烛放在玻璃座里，把火柴放在附近。把蜡烛从左到右依次放置白色、绿色和橙色。

把宝石按你认为有意义的顺序放在蜡烛前面。

例如，如果现在对你来说爱情最重要，那么从左边开始放置爱情石，或者如果健康最重要，那么放置健康石。每次做这个仪式时，宝石的顺序可以改变。

把你的感恩清单放在白色蜡烛下面，把装有常青植物的花瓶放在绿色蜡烛后面，把硬币放在橙色蜡烛后面。

把小碗放在蜡烛的左边

把盘子和涂有蜂蜜和面包以及葡萄酒或果汁的玻璃放在蜡烛的右侧。

从左边开始点燃蜡烛，说：

我在生活中有爱和友谊

点燃绿色蜡烛并说：

我拥有强健健康的身体和清晰的头脑

点燃橙色蜡烛并说：

我的生活中有成功和繁荣

按顺序说出以下句子，每次用右手拿起一颗宝石并放入左手，说完每句后轻轻吹一下宝石然后放入碗中。

我在生活中值得爱和友谊

我应得一个强壮和健康的身体，头脑清晰

我值得在生活中拥有成功和繁荣

我感受到爱和友谊在我周围流动

我感受到身体中健康的流动和头脑清晰

我感受到成功和繁荣在我周围流动

我接受爱和友谊增加到我的生活中

我接受有更强壮和健康的身体以及更清晰的头脑的增加

我接受成功和繁荣增加到我的生活中

在碗中有九块石头后，把面包蘸蜂蜜然后说：

我的生活中有很多甜蜜！

吃面包和蜂蜜。

拿起葡萄酒并说：

我的生活充满喜悦！

喝葡萄酒。

让蜡烛自然熄灭。

拿走放有面包和蜂蜜盘子以及葡萄酒杯，但是把其他东西留在祭坛上接下来的3天。

在第三天，把硬币包裹在你的感恩清单中，把它和常绿植物带到你喜欢的树下埋入树的根部，然后把常绿植物放在顶部并说：

所以我给予我丰富获得的东西！
