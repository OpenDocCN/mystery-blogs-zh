# 黛安娜的礼物：放下过去的悲伤

评分5/5星

当您因与他人的过往经历而仍感到悲伤，并且在沟通和行动方面已经尽了一切后，请来找黛安娜。

**你需要以下物品执行此咒语：**

+   花瓶里的白色花朵

+   七支白色蜡烛在蜡烛台上

+   一把刀

+   金属或陶瓷重碗

+   钳子

+   一把铲子

+   一把硫酸镁

+   笔和纸

+   检查和检查煤

+   香2份没药

+   半份肉桂

+   干薄荷1份

## 施放《黛安娜的礼物：放下过去的悲伤》的指示

圣坛：将花朵，蜡烛和香炉以七支蜡烛呈“半菱形”图案排列，半菱形的尖端指向圣坛的后部。其他所有物品都放在圣坛的一侧。

在月亮渐亏的夜晚，次日清晨继续。

在点燃蜡烛的部分，您需要参与并回应女神的命令，参与所需的内在工作。这是一项深入的过程，是女神和您之间的事情。请耐心等待您的部分。

仪式的身体：在点燃的香炉煤上放一点香。等待烟雾升起，然后说：

"女士，我的悲伤很多；

爱的无辜礼物，

和天堂的花园，

在他人手中受到打击和压碎。

信任的手，

完美人类爱情的破碎幻想，

这些是记忆中的黑暗池塘

我祈祷的事物"。

在煤上撒更多的香，然后说：

"我已经尽了我的能力，

写作，交谈，说出真相。

我已经尽了我的能力，

消除我心灵的痛苦。

但我仍然受伤了

但痛苦仍然在我的记忆中升起。

我需要一个比我更强大的人，

因此，我祈祷，黛安娜，请来治愈我的灵魂"

在煤上撒更多的香，然后说：

"哦，地球和天堂的女王，

希望和勇气的女主人，

魔法的治愈和修复女士，

听我的祈祷。

帮助我释放这些过去的悲伤。

放下我的重担是我的愿望。

因此，我祈祷，黛安娜，请来治愈我的灵魂"。

在煤上撒更多的香，然后说：

"女士；即使在阴影中，希望也会上升。

因为你现在站在我面前。

你的色彩斑斓的光渗入我的灵魂

当你敦促我脱下我的陈旧盔甲时。

与我说话吧，作为万物女神的女士"。

听取她的话语，当她讲话时，让它们进入你的心灵：

"我，黛安娜，在我的神性内怀有，

公正的天平

以及改变之剑。

倾听我的话语和

我会带给你疗愈和平静。

让我的命令深入你的灵魂，

光明将进入黑暗。

因为我是黎明，晨光之母，

我将更新你的灵魂。

"在爱与力量中，我说"

勇敢一点，并为改变做好准备，

知道阴影的时光已经结束了。"

说：“万岁，黛安娜”，并低头致意。在这个仪式的下一部分，回应她的命令。根据需要回答每个问题所需的时间。看见女神在你面前。你将点燃七支蜡烛，每支代表一个命令。点燃最左侧的第一支蜡烛。听黛安娜说：

"拥抱你经历的记忆。

不要害怕。"

探索那些仍然困扰你的记忆。回忆它们，感受它们。从每个情况中写下关键词。点燃最左侧的蜡烛。听她说：

"让每一次伤害从黑暗的池中升起，

我会从你身上带走它们。

再次回顾每一个情况。对于每一个伤害，让痛苦像云一样升起。看着它从你身边飘走。让黛安娜接过它，将它转化为光，并散发出去。这是重要的工作，不要匆忙。看见，感受，释放。根据需要添加更多的香料。

点燃第三支蜡烛。拿起金属或陶瓷碗和你写在上面的纸。握住纸的一角，点燃角落。让它燃烧，尽可能长时间地持着；然后把它放入碗中。如果还有一小角未燃烧完全，用镊子捡起并重新点燃，或者如果非常小，放在熏香炭上。

将香料洒在炭上，点燃第四支蜡烛，听黛安娜说：

"在我的休憩和更新之池中沐浴"。

想象一个美丽的林间小径，有一个小湖或水池。感受自己滑入这片舒适的休憩和更新之水中。在这些水中漂浮，让你的思绪保持空白。点燃第五支蜡烛。再次听她说道：

"相信你的善良和力量。

原谅你的不完美。

看着自己从更新之水中升起。原谅你的不完美。感受你的善良和力量开始从你身上辐射出来。点燃第六支蜡烛。

"理解他人的软弱。

原谅他们的不完美"。

想起那些让你痛苦的人。原谅他们的不完美。将他们交到女神手中。点燃第七支蜡烛。听她的声音：

"很快就要上床睡觉了，进入宁静的睡眠。

我今晚会看着你，

"哦，勇敢的灵魂，充满勇气。

去睡吧，愿你有美梦相伴，

非常尊敬的信徒"。

立即在仪式结束后，用放有少量硫酸镁的浴缸洗澡。让蜡烛燃烧完毕，然后上床睡觉。第二天早晨，拿着铲子去室外挖一个洞。将烧掉的纸灰和熏香炭的残余物埋入其中。覆盖土壤，并用脚牢牢踩实，说道：

"火化为灰烬，

灰烬埋在大地里，

黛安娜，晨光，

"我，重生之曙光！"
