# Regeneration Assist Potion

此咒语允许您制作一种帮助您疗伤的药水。（不能完全愈合）

## Casting Instructions for 'Regeneration Assist Potion'

**您将需要以下物品来完成此咒语：**

+   手部脉轮

+   少量水

+   集中思想

+   带盖/软木塞的瓶子

**您将需要以下物品来完成此咒语：**

+   手部脉轮

+   少量水

+   集中思想

+   带盖/软木塞的瓶子

1\. 用少量水装满瓶子。确保水不要太多也不要太少。只需足够让水在玻璃底部积聚即可。

2\. 将手的中心（手脉轮所在处）放在玻璃口的正上方。

3a. 必要时冥想。

3b. 冥想时，将能量集中到瓶口上的手脉轮。专注于通过手脉轮为瓶子注入治愈能量。

4\. 当完成时，迅速将软木塞回瓶口，以便将能量封存在内。

5\. 能量有时会泄漏，因此最好在封装后的3到5分钟内饮用。

这种药水可能会有一些副作用，比如

麻刺感

轻微灼伤

震动（不危险）
