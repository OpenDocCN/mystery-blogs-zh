+   圣水

+   盐

+   大型石英水晶

+   一件护身符或其他可保护你免受恶灵侵害的物品。

+   一把锤子或其他用于粉碎石英水晶的工具。

+   一把铲子或其他用于挖洞的工具。

+   圣水

+   一把锤子或其他用于粉碎石英水晶的工具。

+   一把铲子或其他用于挖洞的工具。

在你和被恶灵附身的人周围画一个盐圈。

用圣水在被附身者的冠状、第三眼、喉咙、心脏和太阳神经 plexus 脉轮上画一个十字。

确保在盐圈内放置石英水晶。

接下来通过高呼召唤太阳之神 Anyanwu：''我呼唤太阳之神 Anyanwu，我请你帮助我，驱逐这个恶灵并将其送回原处，封印在石英中，如此成定，我请求！''

接下来，一旦恶灵离开附身者，请他离开盐圈。接下来，你也将离开盐圈。

现在，灵魂被困在盐圈内，与石英水晶一同。

接下来，你将请求太阳之神，通过以下咒语将恶灵驱逐至石英水晶中：

''以光明，以太阳，以光之神，Anyanwu，我们请求你来打击他们，并将他们困在石英水晶中，直至永恒，如此成定！''

现在一旦你觉得安全可以回到盐圈内，拿起困有恶灵的石英水晶，并感谢太阳之神的帮助，关闭盐圈。

现在外出并找一把锤子或其他用于粉碎石英水晶的工具，并用以下咒语将石英粉末埋入地中：

''以大地将此灵魂分解为尘土，以火，以土地，将你驱逐回最初之处，永世流转，如此成定！''

接下来将整个盖住，并永不再揭开。
