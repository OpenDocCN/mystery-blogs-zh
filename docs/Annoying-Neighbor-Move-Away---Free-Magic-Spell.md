# 令人讨厌的邻居远离

4种不同的咒语，让某人远离。所有这些咒语都会导致一个令人讨厌的邻居离开你

**你需要以下物品来实施这个咒语：**

+   羊皮纸

+   已选中的人的照片（如果可能）

+   醋

+   带盖小罐

+   绿色蜡烛

+   普通泥土或坟墓泥土

## [令人烦恼的邻居远离]赋予指导说明

**简易远离咒语 #1**

当月亮处于残月相时，在白色羊皮纸上写下你想要搬走的人的全名，以及出生日期。将纸卷起来，如果有照片的话也一同放进一瓶醋里，然后扔进流水体中，想象着你的敌人随着瓶子被冲走而搬离。

**远离咒语 #2**

弄一个小瓶子或者带有盖的瓶子。在一张纸上画一幢有大大的X标记的房子。然后在图片中的房子外面写下他们的名字9次。用4类泰夫斯醋把瓶子灌满，盖上盖子，扔进流动的水体。

你也可以在他们外面的门把手和门廊上泼溅战争水来加强这一效果。当你这样做的时候……想象着他们收拾行李，开车离开……离开他们的房子驶向远方。

**走远咒语 #3**

找到一支绿色蜡烛，足够每天燃烧30分钟，至少连续一周。编写一个简短的咒语，类似于下面这样：

"令人讨厌的邻居离开，

找到一个更好居留的地方。

一处你会很快乐的地方，

我将会很快乐，

全方位你都会很快乐。

一个合适的地方，一个美好的地方，

一个更好居留的地方

只要它离我和我的家远去。

令人讨厌的邻居去！

最好自己创作一个咒语，尤其是如果你可以把它个性化给特定的人。只需要让它积极向上。这里的想法是，你在帮助这些人找到比他们现在拥有的更好的东西，同时也帮助自己获得和平与宁静。

你还需要纸张和一些可以画画的东西。

每晚点燃蜡烛（首先设立一个圈，如果你需要的话），设想火焰是他们心中的一丝火花，将他们带向新的家。然后坐下来，凝视火焰5-10分钟，专注于你的愿望，希望他们找到一个新的，更好的住处，在那里他们会很快乐，并且不再惹人讨厌。

然后，拿起你的笔和纸，在念咒语的同时画出来。简单的图画也可以。在第一晚画出你邻居的（当前）房子和他们所做的让人讨厌的事情。即使图画简单也要画得完整。放进树木，窗户之类的细节。完成后，等待30分钟结束，专注于蜡烛火焰和你的目标，折叠纸张并撕成小碎片。熄灭蜡烛（如果你有勾画出一个魔法圈的话请一并关闭）。保存撕碎的纸屑。

在接下来的一周每个晚上，重复这个仪式，但形式会有变化。第二天，画出他们兴奋快乐的身影站在他们当前的房子前 -- 因为他们找到了更好的房子。在接下来的一周里，画出他们拿出当前房子的各种东西（仿佛他们在搬家）。关键是只展示他们的当前房子。你不想通过为他们画新的房子来限制可能性。

一周后，把所有纸屑和一部分烛尾装在信封里，埋藏或藏在他们和你之间的产权线上。如果你用了大张纸，你只需要包括一部分，以便尽量保持信封的大小合适。

**远离的咒语 #4**

这个咒语需要在弦月期间使用。日期和时间都不重要。收集所有你需要的东西。把土倒入到碗里。把蜡烛放在泥土的中心。（如果有必要可以用一个蜡烛台，长蜡烛可以自由站立）。点燃蜡烛，专注于你想让搬离的人。重复以下咒语三遍：

“远离遥远之处的沙子

名字会离开而我会留下

我们两个，都会很幸福

让名字离我远去。”

连续七天向某人送去一部分沙子。你可以用邮件，但周日你必须亲自送去。你不必亲自把沙子交给对方，只需将其塞进邮箱或靠在他们的门上。
