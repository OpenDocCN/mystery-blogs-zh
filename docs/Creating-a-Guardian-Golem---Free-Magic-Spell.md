# 创造守护者傀儡

这是一个我愿意分享的古老家族咒语，因为外面有一些魔法无法影响的生物，也没有太多的魔法使用者练习自卫了。

## '创造守护者傀儡'的施法说明

**为此咒语你将需要以下物品：**

+   盐

+   白粉笔

+   你想用来制作傀儡的材料（我自己更喜欢使用石头）

+   4 根你将要使用的元素颜色的蜡烛

+   对你追随的神明的坚定信仰

+   小奉献，不能是活的，但食物或葡萄酒可行

+   1 颗代表你的颜色的宝石。

首先最重要的是收集你的材料。

将木头，石头，钢铁或泥土塑造成傀儡的形状，并将宝石放在心脏位置，并在通往宝石保持区的狭窄通道中放置宝石。

使用白粉笔和盐来驱散恶魔占用，画一个内部邀请圈和外部保护圈，遵循你追随的神明的指导。
