# 免费防护咒语

评分为5/5颗星

一个简单的保护咒语。

**你需要以下物品来进行这个咒语：**

+   4块黑电气石

+   1张大纸或者卡片，在上面画了一个大五芒星

+   5支短白蜡烛

+   4个直径约½英寸的小反射片

+   1个小立镜（例如一个旧的粉盒镜）

+   1支可以写在玻璃上的记号笔

+   1支短黑蜡烛

+   1根别针

+   1把类似于女巫巴恩或颠茄的保护草药

+   1个白塑料袋

## '防护敌意者'施法说明

拿起你的黑电气石，把一块放在四个方向的每一个方向上，如果需要的话，直接把它贴在你打算施展咒语的房间的墙上。把五芒星放在你的祭坛中央。划出你的圆圈。

用别针在黑蜡烛上刻下你敌意者的名字。将其放在五芒星的中心。在五角星的每个角落都放上一个带有小镜子的白蜡烛，对准黑蜡烛。在星形的第五个点上，放一个小立镜，上面贴有太阳、月亮和星星的符号。

点燃第一支白蜡烛说：

"我从（方向）召唤光芒，借助这光芒，我排斥任何可能从**姓名**那里接触到我的黑暗。并借助这面镜子，我呼唤三法则，将他/她可能发送给我的能量三倍反射回去。"

点燃剩下的三支蜡烛，每次重复你的诗句。用第五支蜡烛召唤中心、上面和下面，说：

"在太阳、月亮和星星的光芒下：我排斥任何可能从**姓名**那里接触到我的黑暗。并借助这面镜子，我呼唤三法则，将他/她可能发送给我的能量三倍反射回去。"

把第五支蜡烛放在镜子后面，点燃它，然后说：

"姓名，由七个方向，即地球、空气、水、火、太阳、月亮和星星，你现在发出的一切都将反射回你自己。"

点燃黑蜡烛

"你存在，但你的思想不触及圈外的任何人，当这根蜡烛燃烧殆尽时，你所有的影响、所有你带来的苦恼，都会逐渐消失。"

让蜡烛完全燃尽，并非常小心地观察它们，确保它们不会点燃纸张或者把蜡弄脏。在必要时移除立镜。当蜡烛全部熄灭时，用保护草药的灰围绕着蜡烛融化的地方，然后说：

"你的影响力已经衰落至无，你被一圈灰烬所限制和定义。你的力量反射回自己，不能伤害灰烬圈之外的任何人。我召唤守护塔的守护者见证，我站在你的灰烬圈之外，因此超出你伤害的能力。围绕我的黑暗已经随着蜡烛的燃烧而烧尽，现在在太阳、月亮和星星的光辉下，我站在自由之地。"

小心地折叠或卷起纸张，将所有的蜡、灰烬和小镜片包裹在内。放入塑料袋中并系紧口。将其拿出并放在离您家几个街区远的垃圾桶中。

或者，您可以像上面那样卷起纸张并在其上建起篝火，完全销毁它。
