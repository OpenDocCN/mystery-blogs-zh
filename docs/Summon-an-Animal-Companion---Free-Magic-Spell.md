# 召唤动物伴侣

评分 5/5 星

此咒语将创造一种只有您可以看见和触摸的任何类型的动物。您的动物可以具有元素能力以及一项次要能力

## 召唤动物伴侣的施法指南

**为此咒语您需要以下物品：**

+   2张纸

+   铅笔

+   钥匙或具有意义的某个物品

**为此咒语您需要以下物品：**

+   2张纸

+   铅笔

+   钥匙或具有意义的某个物品

在第一张纸上用铅笔画出您选择的动物。要详细描述，使其看起来像真的。但实际上动物不一定真实存在。因此您可以画龙、凤凰或任何其他事物。只需使其看起来有些真实即可。不需要完美，但要画得好看。

现在，在第二张纸上，写下您的动物的：

+   名称

+   性别

+   眼睛颜色

+   元素

+   物种

+   能力

+   个性

现在写下以下内容：

“我呼唤我的新生伴侣（动物的种类），（动物的名字）。他/她将与我同在余生。用我选择的（钥匙/物品）来约束他/她。这是我的意愿，如此成就！”然后大声说出来

然后将纸折叠两次，用大写字母写下动物的名字和物种。再次折叠并用大写字母写下它的元素。然后高喊：“我召唤我的伴侣（动物的名字）留在我身边，直到我余生，如此成就吧！”

现在，整天都要随身携带这张纸。在得到您的动物之前，您需要随身携带您的钥匙或选择的物品
