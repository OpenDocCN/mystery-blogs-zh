# 为物品施加好运

此法术将允许你为一个物品/服装（最好是戒指或项链）施法，使你在它周围时能带来好运。

## 施法说明：'为物品带来好运'

**你将需要以下物品执行此法术：**

+   你想要施法的物品

+   大白蜡烛

+   红色蜡烛

+   粉色蜡烛

+   绿色蜡烛

+   香和香炉

+   匹配

**你将需要以下物品执行此法术：**

+   你想要施法的物品

+   大白蜡烛

+   红色蜡烛

+   粉色蜡烛

+   绿色蜡烛

+   香和香炉

+   匹配

1\. 点燃香并放入燃烧器（注意：必须使用匹配）

2\. 把一个大白蜡烛放在你面前。

3\. 将红色、粉色和绿色的蜡烛排成三角形，并点燃它们。

4\. 拿起你的物品，重复多次地吟咏：

''祝福这个（物体名称）作为魔法的象征，

保护我免受悲惨的灵魂。

带给我好运和财富，

带给我好运，如此定夺！''

5\. 戴上或拿起你的物品，聚集能量制造一个气球，但你实际上不需要制造一个。

6\. 说，“好运”，并想象你的光环进入物品。

7\. 继续持有/佩戴你的物品几分钟，然后吹灭蜡烛。

8\. 将香放在一旁让它燃烧完。

如果你愿意，你可以把香放在紫晶附近。

如果您有问题，可以给我留言。可能需要尝试几次才能生效。
