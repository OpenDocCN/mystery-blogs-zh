# 咒书/保护印章

一个用来创造印章的咒语，它应该保护你的咒书或影子之书免受邪恶的灵体侵扰。

## '咒书/保护印章'的铸造指南

**你将需要以下物品来进行这个咒语：**

+   咒书/影子之书

+   橡树皮屑

+   研钵和杵

+   橡树皮

**你将需要以下物品来进行这个咒语：**

+   咒书/影子之书

+   橡树皮屑

+   研钵和杵

+   橡树皮

首先在你的咒书/影子之书的第一页上画一个你感觉舒适的正立五角星。

然后在五角星的正中间放足够的橡树皮屑，以便完全覆盖它。这意味着你必须尽量均匀地把它散开，同时仍然只有灰尘在中心。

之后，你需要将你主手食指离尘土2-3厘米。如果愿意，也可以使用你的魔杖。然后念道：

"我请求那些想知道我的秘密的人远离我。我请求那些想知道我的咒语的人保持距离。我请求来自强大树木的灰尘保护我的书，直到我移除这个五角星。我请求如此，请让这个成为现实。"

如果你缺少橡树皮屑：首先拿出你的研钵和杵，放入你满意的树皮。然后将树皮磨成粉末。这个粉末可以是细粉末或粗粉末，以你感觉最舒服的方式为准。如果你已经有了所需的所有灰尘，就从画五角星开始，然后继续咒语。
