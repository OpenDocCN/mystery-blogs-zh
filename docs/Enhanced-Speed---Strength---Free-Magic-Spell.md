# 增强速度和力量

两个不同的咒语。一个增强速度，另一个增强力量。

## 施展'增强速度和力量'的说明

**你将需要以下物品来施展这个咒语：**

+   水

+   一个碗

+   一根鸟羽毛（清洁）

+   两滴马血

+   能量饮料

+   一根狼骨（任何小的）

+   两滴狼血

+   水

+   一个碗

**速度咒语：**

用羽毛在碗中搅拌水，同时念着："灵魂啊，请你增加我的速度。我想跑得快。比大多数人都快。这会实现吗。" 将血滴入"马的血，威严，强壮，快速。灵魂啊，请你赐予我这匹伟大骏马的速度。让我像风一样快速奔跑。如果需要更快，更快。我需要这个咒语是因为（告诉灵魂你为什么需要它）听到我的请求。这会实现吗。" 喝下混合物。24 小时后你会更快。

在碗中混合能量饮料和水。用骨头搅拌。念咒语。" 能量和力量是我需要的，灵魂，用这只狼的骨头，我吸收他的力量。我希望变得更强壮，因为（告诉他们原因）" 加入血液。" 一滴巨狼的血，凶猛，聪明，强壮。用这血，我吸收他强大的力量来提升自己。灵魂听到我的请求并让此成真" 喝下混合物。24 小时后你会更强壮。
