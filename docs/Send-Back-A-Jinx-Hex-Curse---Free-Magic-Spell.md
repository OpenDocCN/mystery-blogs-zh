# 将咒、符或诅咒送回

评分为5/5星

一种简单的咒语，可以将轻微的咒、符或小诅咒送回。对于重大诅咒的逆转则无效。

## 执行“将咒、符或诅咒送回”的指示

**你需要以下物品执行这个咒语：**

+   魔杖/法杖/魔法手指/等等

+   白色蜡烛（们）

+   黑色蜡烛（如果诅咒涉及健康问题）

+   不是盈月阶段的月相

+   （可选）目标人物的照片或某种DNA样本。

+   （可选）金缕梅

像平常一样布置你的法阵。点燃白蜡烛（健康问题时也点燃黑蜡烛）。说：“我被施了咒，显而易见，而某某把它送给了我。按照自然法则，我将以三乘三乘三的方式将其回送。”重复三遍。

第三次，握住照片/样本并用魔法手撕碎或捏碎。如果是照片、签名、一缕头发等，请烧毁。如果是体液，则与金缕梅混合以净化，并倒入魔法废物孔。

如果有结束仪式，请结束咒语。如果没有，请简单地清理并关闭法阵。
