# 保护一个物品/物体免于他人干扰

与其他一些物品保护法术不同，这个法术将仅保护物品免受他人干涉。

## 在「保护/阻碍他人触碰物品」的咒语指南。

**你需要准备以下物品：**

+   3个碗

+   4根蜡烛

+   水

+   [厨房] 盐

+   廉价的喷雾器

+   一个魔幻的圆圈

+   一声音

**你需要准备以下物品：**

+   3个碗

+   4根蜡烛

+   水

+   [厨房] 盐

+   廉价的喷雾器

+   一个魔幻的圆圈

+   一声音

准备一碗水、一碗盐和一个可以在药店买到的廉价喷雾器。把蜡烛放在四个角落并点燃它们。

画出一个圆圈并召唤角落里的能量。然后把双手放在水和盐的碗上，让你的魔法通过手臂流向物品，并用保护的光环围绕它们。然后唱这个咒语：

> 保护不被窥视
> 
> 只有我能看到
> 
> 保护不受窥视
> 
> 这些法术的本质
> 
> 制作一个神秘的面纱
> 
> 保护这个魔法（你的物品/物品的名字）的核心
> 
> 让我们的界限变得模糊
> 
> 混合、抹去和模糊
> 
> 只有我的眼睛能看到
> 
> 被委托的秘密

然后把手放在水上，唱:

> 我为这个水祈福，保护它
> 
> 以女士和男士的名义
> 
> 如此成就吧！

然后把手放在盐上，唱:

> 我为这个水祈福，保护它
> 
> 以女士和男士的名义
> 
> 如此成就吧！

然后把水和盐放进喷雾器中，轻轻喷洒你的物品，那么你的个人物品就得到了保护！
