# 找到并召唤你的灵兽

## 施法说明 '找到并召唤你的灵兽'

**你将需要以下物品来进行这个咒语：**

+   纸

+   笔

+   玩具动物：狐狸、狼、狮子、鹿、猴子、蛇、熊、鸟、马，

+   水

+   水

拿起笔和纸。从下面的列表中，写下你是其中之一的特质（说实话，否则它不会起作用）：聪明、关心、冒险、冒险、快速、友好、安静、深思熟虑、谦虚。确保你只写下一个！现在将这张纸折叠起来，把玻璃杯拿到阳光下，诵读这段话：

"伟大的灵兽

他来找我了

为我带来伟大

在你身上我看到

最好快点

显示出很大的匆忙

进入我的心

我不会浪费给你

我向你鞠躬，

我的特质是（在这里插入特质）

依赖于我

因为是你，灵兽，

我所寻找的。"

现在喝下这杯水。回到屋里。把所有的毛绒玩具拿出来排成一排。把折叠的纸放在它们前面，在纸上写着：“给我的灵兽”。然后诵读这段话：

"聪明的狐狸，可能是

或者关心的马就是我

也许是冒险的熊

或者冒险的蛇，

急迫的狮子

友好的狼

或者安静的鹿

深思熟虑的猴子，

谁敢不轻视

也许是谦虚的鸟

不言而胜

它可能是谁呢？

我们会看到..."

现在去睡觉，当你入睡时，专注于你的特质。在你的梦里，这种动物会出现在某个地方（很可能在你附近）。它可能会和你说话，也可能不会。记住这个动物。这就是你的灵兽。
