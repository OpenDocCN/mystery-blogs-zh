# 在法律案件中揭示真相

确保法律案件中揭示真相；专为那些被错误审判/针对的被告设计。请注意，不要用于试图操纵案件，而自己实际上是错误的情况。

## ‘揭示法律案件中的真相’施法说明

**你将需要以下物品进行这个咒语：**

+   龙血香

+   黑蜡烛

+   黄蜡烛

+   原告的名字写在一张纸上

+   祝圣水（月亮或暴雨水都可以）

+   黑盐

**你将需要以下物品进行这个咒语：**

+   龙血香

+   黑蜡烛

+   黄蜡烛

+   原告的名字写在一张纸上

+   祝圣水（月亮或暴雨水都可以）

+   黑盐

1\. 如果这是你的实践的一部分，请圈定

2\. 点燃龙血香并沉思需要澄清的案件事实。

3\. 将对方的名字写在一小张纸上，放在黑蜡烛前面。

4\. 点燃黑蜡烛并说：“在真相中有力量，在真相中有力量”

5\. 点燃黄蜡烛，并对着黑蜡烛说：“我被真相所保护，事实站在我的一边。”

6\. 把纸条通过熏香烟雾，说：“在风的名义下，你的谎言无力。”

7\. 用黄蜡烛点燃纸条，让它在你的大铁锅或铸铁锅中燃烧（为了安全）。说：“以火的名义，你的谎言无力。”让纸条烧掉

8\. 将暴雨水倒在灰烬上，并说：“以水的名义，你的谎言无力。”

9\. 撒上黑盐，并说：“以地的名义，你的谎言无力。”

10\. 最后说：“以元素的名义，真相将会显现。我站在光明中，与宇宙同在，我受到保护。如此成就。”

11\. 如果这符合你的实践，请解开圆环。清除铸铁锅中的残留物并妥善处理。如果你要去法庭，而你使用了一小块蜡烛，你也可以带着它去法庭。
