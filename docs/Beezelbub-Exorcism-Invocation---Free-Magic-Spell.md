# 比利吉驱魔祈愿

这个咒语引导了比利吉的精神本质进入女巫或从业者的体内，通过咒语来驱逐或移除一个容器、物品或空间中的负面能量。

**完成此咒语需要以下物品:**

+   中心标有一个大黑色三角形的请愿书

+   任何种类的一滴血/唾液；或橄榄油

+   比利吉的名字写在另一张黑墨水的请愿书上

+   一个白色或深色的碗（如果不是白色则最好是橙色）

+   神剑或法杖

+   圣水（月光水、阳光水、鼠尾草水等）

## “比利吉驱魔祈愿”施法指南

把第一个中心标有大黑色三角形的请愿书放在你神圣空间的中间。

使用魔杖/神剑召唤基督教的四个主要方向（请勿召唤东方）的精灵/能量

用神剑/法杖在标有黑色三角形的请愿书周围凝结一个能量三角形。

把碗（白色、橙色或深色）放在三角形请愿书上。

用血滴或橄榄油涂抹第二个写有"比利吉"名字的请愿书。

如果你没有第一种两种涂抹液体；用你的舌头滴唾液到“比利吉”的中心。

把请愿书放在你的左手上，并把右手放在上面。

持续视觉化你的心脏脉轮通过你的连接进入持有的请愿书。

说比利吉的真言：

"我呼唤你，比利吉，啊苍蝇和腐烂的尘土之主。让你的灵魂显灵在我里面。这样我就可以驱逐这个不服从的灵魂或你的能量。我呼唤比利吉的灵魂。"

把请愿书放进碗里。

双手持神剑或法杖，刀锋或尖端指向天空。感受或想象心脏脉轮的能量通过神剑或法杖被推拉。开始呼吸并听着你的呼吸进入一种恍惚状态。然后想象从你的神圣空间中的设置中发出潮湿的橙色光芒。进入并驱散空间、容器或物体的负面灵魂或能量。感受到比利吉在你的行为和咒语中的振动。

背诵真言：

"以比利吉之名，我命令不洁净的意图离开我的显现体。"

负面能量的存在将被消除，并应该被提供给比利吉作为报酬。

法术完成并献祭后。取您的神圣或神圣的水，在整个设置和胸部以及右臂和右手上撒洒。将所有请愿书投入火中处置。（不要种植在花园或墓地，因此与贝利尔波比的联系将被打开，可能或不可能在自然界中表现出他/她的愿望。）

如果无法燃烧请愿书，请将其放入垃圾桶并思考放弃愿望的请愿书。

您可以以任何您认为合适的方式清洁您的碗，魔杖/神剑。
