# 将厄运送给另一个人

这是一个相对简单的咒语。我不知道它是否有效，但可能会有效。

## ‘将厄运送给另一个人’ 的施法说明

**你将需要以下物品来施法：**

+   13颗蛋白石

+   4支黑色蜡烛

+   17片枯叶

+   2 汤匙盐

+   魔杖

+   被施加厄运的人的照片

+   6 茶匙檀香粉

+   2 茶匙阿苏木

+   一张写有 13 的纸片。

+   声音

**你将需要以下物品来施法：**

+   13颗蛋白石

+   4支黑色蜡烛

+   17片枯叶

+   2 汤匙盐

+   魔杖

+   被施加厄运的人的照片

+   6 茶匙檀香粉

+   2 茶匙阿苏木

+   一张写有 13 的纸片。

+   声音

1.  将枯叶排列成一个矩形，将蜡烛放在一个正方形内。

1.  用盐撒在蜡烛上。在蜡烛周围放置 11 颗蛋白石。

1.  点燃蜡烛，轻敲被施加厄运的人的图片，放在蜡烛中间。将阿苏木和檀香粉混合撒在蜡烛和图片上。放上写有 13 的纸片并点燃。现在在图片和蜡烛上挥动你的魔杖，同时重复十次：

> "哦，厄运，请来吧，请来吧，
> 
> 听从我的请求，前来（人名）。
> 
> 让他们的生活变得可怕，持续插入多少天、周、月、年的厄运。"

4.    然后将其余的蛋白石放在图片上，静置 25 分钟。然后熄灭火焰。
