# 闪烁水精灵咒语

此咒语的目的是吸引水精灵和有益精灵的神圣洞察。

## 用于“闪烁水精灵咒语”的施法说明

**你需要以下物品来施法：**

+   一个小蛋糕

+   一束鲜花

**你需要以下物品来施法：**

+   一个小蛋糕

+   一束鲜花

在日出前的清晨，即满月前一天或女巫和巫师的女巫节上施法。坐在水域旁边，比如小溪、小河、河流、天然水池、池塘、洄游池、水库、湖泊或海洋旁边。如果你有这个习惯，画一个魔法圈围绕你坐的地方。使用一半的花来勾勒魔法圈。唤入元素力量。通过说出以下话语，邀请水精灵和其他有益的精灵进入你的圈子：

''友好又有益的水妖

祝福我，今天让你的存在降临吧。''

将蛋糕切成两半。将一半蛋糕碾碎并放入水中，向水精灵献上。在这么做的时候，说：

''来自东、南、西、北友好的水妖，

闪耀明亮，为我祝福吧，水精灵

温柔的水精灵，快现身吧

请为我带来魔法的洞察力。''

将另一半花逐一放入水中。在这么做时，重复念咒语：

''闪耀明亮，水精灵

请为我带来魔法的洞察力。''

静静地坐上至少三十分钟，与周围的水精灵交流。向他们询问你寻找答案的问题。当你完成时，感谢水精灵和精灵，并撤去你画的魔法圈。将用来勾勒魔法圈的花朵撒在水上。在这么做的时候说：

''谢谢你，友好的水妖，

感谢你今天的祝福。''
