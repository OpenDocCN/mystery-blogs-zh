# 揭示某人所想

评分 5/5 星

基本上这是一个能够揭示一个人对你的看法的咒语。你将在梦中知道真相。这个咒语可以在任何月相下进行，但最好在盈月时进行。

## 对 '揭示某人所想' 的施法说明

**你将需要以下物品来进行这个咒语：**

+   想象中的人的图片

+   少许干迷迭香（如果你没有迷迭香，可以用莳萝、薄荷或其他用于清晰思维的草药替代）

+   耐火盘

+   一小张纸

画一个圈并点燃蜡烛，用你的意图充满它。为了真相和思维的启示而点燃它。在脑海中清晰地想象这个人的形象非常重要。

在一张纸上写下“我希望梦中揭示这个人对我的看法”。

将迷迭香或者如果你使用任何其他草药的话将其包裹在纸中，使其紧密地捆绑，然后小心地点燃纸并将其放入耐火盘中。当它燃烧时，再次想象那个人的脸，并专注于你想要的东西。

说出以下的话语：

> **“用梦想者的眼睛我寻求看见
> 
> 这个人对我有什么看法。
> 
> 无论是好是坏
> 
> 揭示给我，如我的意愿所愿。**

当迷迭香和纸烧完时，将蜡烛灭掉或让它自然熄灭并入睡。你将在梦中得到对你问题的答案。
