# 夜晚能量加在物体上

这个咒语可以让你选择的物体被夜晚的能量所施加，所以你可以随身携带它们，即使在白天。

## 施放'夜晚能量加在物体上'的指导

**进行此咒语，你需要以下物品：**

+   你希望施加魔法的物体（吊坠、石头等）

+   足够容纳物体的碗。

+   水

+   黑墨水

+   滴管或眼药水滴管

+   黑色丝绸、皮革或棉质袋子，大到足够容纳这个物体。

+   纸巾
