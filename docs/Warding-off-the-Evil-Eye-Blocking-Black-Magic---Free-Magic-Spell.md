# 避邪/阻挡黑魔法

这将有助于避邪。它还将有助于阻止他人对你施展黑魔法。

## 施法说明'避邪/阻挡黑魔法'

**你需要以下物品来施展这个咒语：**

+   迷迭香

+   薰衣草

+   黑胡椒

+   肉桂

+   姜根

+   6支白蜡烛

+   一支黑蜡烛。

**你需要以下物品来施展这个咒语：**

+   迷迭香

+   薰衣草

+   黑胡椒

+   肉桂

+   姜根

+   6支白蜡烛

+   一支黑蜡烛

指示

这个咒语应该在新月相期间施展

首先施展一个魔法圈（向守护者致敬）

面向北方坐下冥想

起身点燃你的蜡烛，顺时针方向，除了黑色的。

然后当你点燃了你的白蜡烛后，用一支白蜡烛点燃黑色蜡烛。

然后点燃草药和香料的混合物。

说：

“无论何种邪恶降临到我这里

我将你赶回去，我毫无畏惧

风的速度和黑夜的黑暗

愿你所有的庇护飞走

以海的迅速

我所拥有的所有力量

如我所愿，即可成真。

然后大声说“我将你赶出去”

尽可能用力吹灭黑蜡烛。

让混合物燃烧，然后在蜡烛上画一只眼睛，用白布包裹起来，埋在你的后院。
