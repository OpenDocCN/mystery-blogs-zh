**摆脱恶魔或负面实体，并恢复其对你脉轮所造成的伤害。（非常高级！）**

第一部分是一种驱邪，第二部分是一种治愈，并将恶魔和其他负面能量远离你，只要你遵循最后的步骤。

**你将需要以下物品来进行这个咒语：**

+   你将需要：

+   -残月。

+   -可能导致此情况的物品。基本上是一件它与你有联系的物品，或者一件你了解的将它与你绑定的物品。

+   （如果你没有这个，找到它最接近的代表。）

+   -2根黑色蜡烛（用于驱逐）

+   -2根白蜡烛（用于净化）

+   -2根灰色蜡烛（用于抵消）

+   （所有这些都应该是持久的、慢燃的。）

+   -一个打火机或很多火柴。

+   选择两到三种以下香料：

+   茉莉花

+   罗勒

+   鼠尾草

+   玫瑰

+   杜松子

+   阿尔塞木（小心燃烧！最好与茉莉花一起使用。）

+   松树

+   注意：选择一根长棒香。每次只需一根。*

+   -防火表面用于所有已点燃的蜡烛。

+   -一块你愿意放弃用来放置或点燃香的防火盘。

+   -黑色布料

+   （以上两种将用于处理绑定的物品。物品的大小将决定盘子和布的大小。尽量不要让它太大或者防火。如果是，我将详细介绍如何处理。）

+   -画一个五角星，你可以坐在中心，用白色粉笔或黑色木炭。

+   -画/创造一个代表你或守护灵的符印。你可以两者都做。

+   用于脉轮的蜡烛：

+   挑选出一种使用蜡烛，相当小且有点长。确保尺寸代表每个脉轮上的损伤。

+   （最长的蜡烛表示最严重的损伤。最短表示没有损伤？选择一个代表所选脉轮的颜色的石头。

+   颜色：

+   红色（根部）

+   橙色（脐部）

+   黄色（脐部）

+   绿色（心脏）

+   蓝色（喉部）

+   靛蓝（第三眼）

+   白色或紫罗兰色（冠部）

+   代表四大元素的物品：

+   地球：一块岩石或岩石，大绿色蜡烛（未点燃），大绿色岩石，各种小绿色岩石，任何代表地球的石头。

+   水：

+   一支大蓝色蜡烛（未点燃），天使石，各种小蓝色石头或河石，一块大河石，任何代表水的石头。

+   火：

+   大红色蜡烛（未点燃），血石，石榴石，小红色石头，大红色石头，任何代表火的石头。

+   风：

+   大黄蜡烛（未点燃）、黄玛瑙、小黄石、大黄石、代表空气或风的任何石头。

+   我的相册里会有一张准确描述如何设置这一切的图片。

## 施法说明：“摆脱恶魔\消极实体，并恢复它对你脉轮造成的伤害。（非常高级！！！）”

确保捆绑的物品在这个仪式开始时离开你的圆圈。

第一部分：驱逐。

1.) 首先点燃黑色、白色和灰色蜡烛。

还不要点燃香。

2.) 离开圆圈并拿起捆绑的物品。（如果它是小的且易燃的。）

3.) 用白色、灰色和黑色蜡烛烧掉它。

（如果是正确的物品，它会很快燃烧或突然燃烧。）

立即将其放在待处理的盘子上，完全用黑布覆盖它和盘子，并点燃香。

用黑布覆盖烧焦的物品。

（它必须比盘子大。）

点燃脉轮蜡烛，并坐在五角星的中心位置完成后。

如果你还没有自己的唱诵，这里有一个好的选择。

唱诵：

远离，远离，远离。

别在今天再来折磨我了。

消散，消散，离开并消散。

飞走，远远飞走。

远走高飞，离开。

从今天起，你将远离。

感受能量的增强，因为你的灵魂从这片黑暗中解脱出来。

唱诵直到你觉得足够安全可以离开圆圈。

第二部分：治疗。

离开圆圈并走出你所在的房间\附近。

等到你确定香已经完全燃尽，并尽力感觉圆圈中的负面实体根本不存在。

（什么恶魔？这个仪式只是为了调整我的脉轮健康！）

这将摧毁它。

在完成这些步骤后，重新进入，熄灭所有蜡烛，并丢弃盘子、布料和所有其内容物。

不要揭开盘子的覆盖物。

拿起布和盘子，特别小心地不要触摸内容物。

不要想你手里拿的东西。

简单地避开眼睛，把它扔到远离家里的地方，或者放在室外的垃圾桶里。

（如果在家附近的垃圾桶里，你必须克制自己，直到垃圾被运到垃圾场。）

你可能注意到在仪式进行过程中你开始感觉更好了。

所有燃尽的有色蜡烛代表已恢复的脉轮。所有你必须熄灭的立着的蜡烛都有更严重的损害，因此需要额外和单独的治疗。

记住这些，祝你一天净化愉快。 :)

撕裂、切割或损坏物品。如果是一个几乎不可能破碎的石头或金属物品\珠宝，当把它放在盘子上时唱诵：

消失，消失，离开，离开。

这现在是你的休息地。

一个逃不出的盘子。

一个盘子和一块布将成为你的坟墓。

尽可能多地投入精力。

如果是一条长项链，用黑、白和灰色蜡烛加热，直到非常热或变红。

将其放在盘子上并点燃香。
