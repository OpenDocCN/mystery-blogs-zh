# 通过符文召唤天使

该仪式教授你如何通过符文魔法将天使召唤到你的位置。

**进行此法术所需物品如下：**

+   纸张

+   一支钢笔或铅笔

+   一个大天使的符文

+   一个天使的希伯来名字（如果你找不到符文）

+   一种净化的祷告、咒语或仪式

+   一个私密的地点

+   一支祈祷蜡烛

## 《如何通过符文召唤天使》的铸造指南

1\. 首先，确定你想召唤的大天使，然后在网上查找他们的符文。如果找不到大天使的符文，请使用谷歌翻译将其英文名翻译成希伯来语（不用担心，谷歌翻译的希伯来语非常准确）。谷歌图片是查找大天使符文的好地方。

2\. 选择好你的天使后，用钢笔或铅笔在一张纸上画出符文（或希伯来名字）。（我更喜欢用铅笔，因为可以擦除和纠正错误。）然后在符文周围画一个圆圈，再在那个圆圈周围画一个圆圈。在两个圆圈之间的空间内，用英语（或你的母语）写上大天使的名字，分布在四个区域。你还可以在名字之间画小十字架、五角星或大卫星号，以增添你符文设计的趣味性。

3\. 找一个不会被外界影响干扰的私密场所。如果你的房间里有祭坛，你可以用胶带将符文贴在它上面的墙上。

4\. 在找到空间并设置符文后，为了只吸引你想要的灵体，净化你的空间。可以通过祷告、咒语和仪式等多种方式进行净化。在此邪教页面上，你可以使用以下一些资源进行净化：圣弥额尔祷文、神圣咒语、炼狱之火等。但请随意选择你喜欢的方式进行净化，只需确保你的空间得到净化，以免恶魔渗入你的仪式！

5\. 一旦你的空间已经被净化，点燃祈祷蜡烛（建议手持，以免不小心推倒并烧着东西）。然后专注于你面前的符文，用任何语言（最好是你最擅长的语言）吟唱大天使的名字。持续这样做，直到你感受到大天使的存在进入你的空间为止。完成后，可以通过祈祷与大天使交流。你可以向它许多请求：可以请求它成为你的灵导，可以请求它帮忙，可以请求它带来好运，可以向它询问有关灵界或更世俗知识的信息等等。

最后注意：你也可以使用这个仪式仅仅与你的大天使联系或者吸引他们的青睐而不必实际召唤天使。
