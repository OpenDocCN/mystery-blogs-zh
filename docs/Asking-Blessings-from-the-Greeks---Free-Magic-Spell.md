# 向希腊神祈福

向一个或多个希腊神祈求祝福或恩惠。

**为了这个咒语，你会需要以下物品：**

+   一些点燃蜡烛的东西

+   至少一支蜡烛

## '向希腊神祈福'的施法说明

在请求祝福或恩惠之前，请确保你知道向哪位神或女神请求特定的祝福。例如：如果你希望某人爱上你，或者希望得到美丽的祝福，阿芙洛狄忒就是你应该去的女神。但是，你不会向战神阿瑞斯请求这种特定的祝福。

每位神或女神的蜡烛颜色（还有更多颜色，但我的资源有限）：

+   宙斯：金色或紫色

+   赫拉：白色、皇家蓝、紫色、玫瑰色、深绿、银色或灰色

+   波塞冬：灰色、蓝色或绿色

+   冥王：黑色

+   狄蜜特：绿色、深棕色、金色、海军蓝或粉红色

+   赫菲斯托斯：红色或橙色

+   雅典娜：金色、橙色、黄色、翠绿色或皇家蓝

+   阿芙洛狄忒：红色、粉红色、紫色、银色、水绿色、淡绿色（海泡石）或任何浅蓝色的阴影

+   阿瑞斯：深红色

+   阿波罗：金色、白色或黄色

+   阿尔忒弥斯：银色、白色、红色、绿色或绿松石色

+   赫尔墨斯：任何颜色

+   狄俄尼索斯：紫色

+   海斯提亚：金色、深玫瑰色、薰衣草色、银色或黑色

+   赫卡忒：黑色、橙色、黄橙色或红橙色

如果你向多位神祈求祝福，请确保每支蜡烛颜色不同，但如果你不得不使用相同的颜色也没关系。

盘腿坐在地板上，把蜡烛放在你面前，或者如果你有足够的蜡烛（用于多位神和女神），你可以把它们摆成一个圆圈站在中间。拿起第一支蜡烛并点燃它。

凝视火焰，清空脑中的所有思绪。然后专注于蜡烛代表的神或女神，热情地问候他们。当你感受到情感联系时，你可以开始说话。（如果这样做让你感到舒服，你可以将蜡烛放在地板上或者在说话时拿起蜡烛。）

没有记住的咒语。只需礼貌地和尊重地请求上帝或女神的祝福/恩惠。解释为什么你想要它。要谦卑和尊重。注意：不要求上帝或女神杀人。

当你完成第一支蜡烛后，用下一支蜡烛重复这个过程。当你完成所有蜡烛的过程后，回到第一支蜡烛。重复你的请求然后吹灭蜡烛。对每支蜡烛都要这样做。

这个咒语可以用于好或坏的原因。如果原因不符合要求，那么你将无法得到你的祝福或恩惠。
