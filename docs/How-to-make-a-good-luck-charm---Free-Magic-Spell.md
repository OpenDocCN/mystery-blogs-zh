# 如何制作好运符

这会制作一个好运符。

## '如何制作好运符'的施法说明

**你需要以下物品来完成此法术：**

+   蜡烛（任何类型）

+   水杯

+   Stone (any type)

+   空罐子（最好是带盖的玻璃罐）

+   薰衣草（刚采摘的）

+   火柴

+   你的声音

+   你的呼吸

+   召唤圈

**你需要以下物品来完成此法术：**

+   蜡烛（任何类型）

+   水杯

+   石头（任何类型）

+   空罐子（最好是带盖的玻璃罐）

+   薰衣草（刚采摘的）

+   火柴

+   你的声音

+   你的呼吸

+   召唤圈

首先，制作你的圈子，然后拿起你的罐子，说出以下的话。

"北风啊，北风我呼唤你，南风啊，南风我呼唤你，东风啊，东风我呼唤你，西风啊，西风我呼唤你，进入这个罐子，我现在呼唤你！"

然后把你的罐子放在圆圈外的任何地方。

现在拿起你的蜡烛点燃它，把它放在圆圈外的任何地方（不要放在罐子旁边）

然后拿起你的水杯，把它装满，放在圆圈外的任何地方（不要放在罐子或蜡烛旁边）。

最后拿起你的石头，把它放在圆圈外的任何地方（不要放在罐子、蜡烛或玻璃旁边）

现在咒语可以开始了

拿起你的薰衣草，点燃（花朵），然后吹灭，把烟团绕圈子转动，说道

“保佑我，保佑我，哦元素们保佑我，将你们的幸福注入我，祝福我的灵魂，成为实现的一部分！无论高低，让世界展示！无论你在何处，保佑我，保佑我，成为实现的一部分！”你可以做到这一点，但你必须保持手在圆圈中转动（顺时针或逆时针），如果烟雾散去，然后点燃薰衣草，再次吹灭。

AFTERWARD

通过打开罐子释放风，不要期望会有大龙卷风或飓风产生，那只是孩子气，把水扔到地上，吹灭蜡烛，拿起石头（它会带给你好运），把你在召唤圈中使用的石头/木棍放在一边，现在它们可以用在其他法术中。”
