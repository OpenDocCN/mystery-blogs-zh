一种自创的使用签和布娃娃进行占卜的方法。

## 施法说明：“用签和布娃娃占卜”

**你将需要以下物品来施法：**

+   一块浅色布和一块较深色布

+   Thread

+   针

+   剪刀

+   带有茎和叶子的花来填充

+   8 个签，每个分类 2 个：

+   2 代表保护

+   2 代表祝福，好运

+   2 代表关系

+   2 代表厄运

+   最后应该有两块骨牌或签来代表它们。它们是：

+   一块有两个 3 的骨牌

+   一块带有数字 5 的骨牌

首先，你需要制作一个布娃娃。布娃娃充当提问者或被问及个体的角色。在轻薄布上画一个人形并剪下来。然后在较深的布上追踪这个形状并剪下来。用几根完整的花枝、叶子和其他东西填满两块布之间的空间。让其中一些超出布娃娃的边缘。它不需要完美。接下来，穿好针线，开始缝制布娃娃，确保不要缝得太紧。完成后打个结并剪掉多余的线头。

现在这个制作完成了，最好手工制作你的签。你不需要为骨牌这样做，但你可以制作代表它们的签。

对于签，我使用了八颗木制方珠和两颗长方形的珠子。对于代表物，使用您感觉舒适的符号。

要施放：

把布娃娃放下，把你的签（lots）放在你合拢的手中。当准备好时，同时把你的签扔在布娃娃周围，并开始解释它们的含义。

你的 8 个签代表生活领域。骨牌各自代表不同的事物：

三块一组- 大笔钱。

一块骨牌 5- 不幸的婚姻，关系。

经过一段时间的实践，你将更好地理解签的含义，以及如何解释它们施放后的位置。
