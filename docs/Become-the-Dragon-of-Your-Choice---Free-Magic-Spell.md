# 成为你选择的龙

评为5/5星

使用此咒语，你可以随意转变成某种龙。如果你使用不同的龙做多个咒语，你可以成为不止一个。

## 施法说明 '成为你选择的龙'

**你将需要以下物品来进行此咒语：**

+   声音

+   对龙的坚定信念

+   项链 - 如果你做多个的话 - （银盘，其余的不重要）

+   选择你独自一人的地方（最好是外面）

**你将需要以下物品来进行此咒语：**

+   声音

+   对龙的坚定信念

+   项链 - 如果你做多个的话 - （银盘，其余的不重要）

+   选择你独自一人的地方（最好是外面）

坐在双腿紧贴地面，项链放在你面前。

把你的手放在上面。想象自己有龙的特征。

唱三遍：

''天空，大地，冰，太阳和海的

我选择我想成为的家族

天空快速可以飞得更高

太阳可以忍受干旱和火灾

地球它强大聪明快速

冰在寒冷中注定要持续

海的法则只能使你自由

（说你选择的元素）是我想成为的那种''

要变成你选择的龙，说：

''我希望变成我的（与之前相同的元素）龙形态''

要变回人类，说：

''我希望再次变回人类''

注意：如果你做多个，你只能每个元素做一个，每个项链上只能有一个龙。
