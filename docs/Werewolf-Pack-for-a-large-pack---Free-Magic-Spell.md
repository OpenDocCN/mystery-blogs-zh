# 狼人群体，一个庞大的群体

一个将一群人变成狼人的咒语。狼型，而不是狼人。开始之前请阅读所有内容。关于“大小”部分，您可以选择两行中的一行，说出狼是正常的，还是暮光之城/恐狼的大小。

+   至少五个人

+   紫色蜡烛

+   皇家蓝蜡烛

+   绿色蜡烛

## 用于“狼人群体，一个庞大的群体”的施法说明

''月亮指引我们，

让它成为现实。

让我们成为月亮的孩子。

我们将成为强大的狼，

我们强大的爪子、钝爪和锋利的獠牙

将击倒我们的猎物。

我将领导我们的群体，

作为阿尔法狼。''

然后贝塔跟着吟唱：

''我们将在夜晚变身，

成为伟大的狼，

（与普通狼相同大小/与人类肩膀高度相同）

野性，自由。

变身将是迅速而无痛的，

我们可以随时激活它，

或者在我们希望以人类皮肤行走时结束它。

我们将完全控制。

我们不会除非出于自愿而杀戮，

我们将自由奔跑。

我们将拥有我们的狼感知能力，

力量和速度，以及敏捷，

在我们两种形态中。

我们还将在两种形态中拥有彩色视觉。

自由将属于我们！

我将协助我们的阿尔法，

作为贝塔狼。''

三角应该说：

我们将为我们的群体服务，不会让彼此陷入危险之中，月亮女神赛琳娜用这些月亮石充满我们狼人的能力，我将训练并领导狩猎，作为三角狼

其余的圈继续，从贝塔左边的人开始，依次绕着圈子。他们一个接一个地说这个咒语：

''我自由奔跑，我狂野奔跑，

月亮是我的指引之光。

我们超越人类，

与大地合为一体。

地球、空气、水和火，

我们的世界，我们了解你。

驱使我们前进，让我们成为，

美丽的狼！

让它成为现实。''

然后阿尔法，然后贝塔，然后其他人按照之前的顺序说这个咒语，然后烧掉图片。

''我将变成这只狼，

图片中的那个。

我的皮毛将闪烁着它的颜色。

我将随意变化，前后不一，

并像月亮一样升高。

这是我的意愿，

愿如此成真！''

一旦图片被烧毁，每个人都举起宝石或水晶，想象自己作为图片中的狼奔跑。当他们有一个清晰的画面时，他们把水晶放下，在蜡烛上刻下自己的名字，下面是一个五角星。

当每个人都完成时，向圆圈中心鞠躬。拿着刀和水晶离开，让蜡烛燃烧到底。
