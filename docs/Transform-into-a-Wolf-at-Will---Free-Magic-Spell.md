# 随心所欲地变成狼

这不是一个兽人咒语，只是一个可以将你变成狼、郊狼、豺狼和狐狸品种中任何一种的变形咒语。

## '随心所欲地变成狼'的施法说明

**你需要以下物品进行此咒语：**

+   一只咒语中任何一只狗的小玩具（或几只玩具）

+   红色蜡烛

+   黄色蜡烛

+   蓝色蜡烛

+   橙色蜡烛

+   深绿色蜡烛

+   一块糖或类似物品

+   夜晚

+   湿/潮湿的毛巾

+   声音

+   想象力

+   信念

+   对咒语中任何一只狗的喜爱/爱好

+   一个碗或盘子

+   一只咒语中任何一只狗的小玩具（或几只玩具）

+   红色蜡烛

+   黄色蜡烛

+   蓝色蜡烛

+   橙色蜡烛

+   深绿色蜡烛

+   一块糖或类似物品

+   夜晚

+   湿/潮湿的毛巾

+   声音

+   想象力

+   信念

+   对咒语中任何一只狗的喜爱/爱好

+   一个碗或盘子

1\. 将湿毛巾放在你面前的地板上。

2\. 将蜡烛按照这个顺序放在你面前：橙色、红色、蓝色、深绿色、黄色。

3\. 点燃蜡烛，将碗/盘子放在它们前面。

4\. 把玩具和糖放在碗里。

5\. 想象一下你最想成为的狗的模样大约5秒钟。

6\. 吃掉糖（但不要在念咒语时吃），同时摸着玩具。

7\. 说一次这个咒语：

> "我想变成这些狗，
> 
> 我想成为像狼一样骄傲和强壮，
> 
> 如同狡猾灵巧的郊狼一般，
> 
> 如同狐狸一般聪明和蓬松，
> 
> 如同豺狐一样美丽和足智多谋。
> 
> 当我嚎叫时，
> 
> 当我尖叫时，
> 
> 当我吠叫时，
> 
> 当我咆哮时，
> 
> 我变换我的形式。"

7\. 念完咒语后，吹灭蜡烛。

8\. 等待。这将需要一个星期，要测试是否奏效，请说：

"我将化身为一个（说出一种具体的狗的品种，如黑背豺、红狐等）"

要变回来，想着“我的真实形态退缩，人类归来。”

在第一次换班后，如果想再次换班，只需单独低语你想变成的东西的名字，与句子分开。
