# Sahia灵魂显现

一种可怕的不可见的死亡蔓延在这片土地上。这是肉体中的灵魂的死亡。Sahia拥有精神和恶魔，她用甜美的声音低语所有的一切都将被吸引到她身边。她几乎摆脱了你的束缚。生命会离开肉体，死亡将像某种渴望活人的不死瘟疫一样蜂拥而至。上帝的殿堂无法为这个恶魔提供庇护。无论你站在哪里，她都会来索取你。上帝的殿堂将被摧毁，邪恶将在这片土地上肆虐。

**你需要以下物品来咒语：**

+   1个木制祭坛

+   1个祭坛碗

+   2 张折叠两次的页面

+   4支白色蜡烛

+   1支小红蜡烛

+   1支小黄蜡烛

+   1支小绿蜡烛

+   1支小蓝蜡烛

+   1把权力刀

+   1 把凳子

## "Sahia灵魂显现"咒语施法说明

说明

(1) 祭坛应该面向北方。把祭坛碗放在祭坛的中央。把黄蜡烛放在祭坛碗后面。红蜡烛放在祭坛碗前。把蓝蜡烛放在祭坛碗的左边，绿蜡烛放在右边。把权力刀放在祭坛的右边。在围绕仪式圈的四个方位放上白色蜡烛。

(2) 站直，高举权力刀。想象宇宙能量击中权力刀并进入你的身体，给你充电。让自己充满这股能量，并把它留在自己的内心。

(3) 降低你的手臂，让权力刀指向前面。说出咒语的线，并吸收那个方向元素的能量，并用同一方向的蜡烛点燃祭坛上的蜡烛碗。顺时针转到下一个方向，做同样的事情，直到你再次面对北方。

(4) 坐在凳子上；闭上眼睛，让你的思绪平静下来，不要专注于任何一个。睁开眼睛，请求保护，并再次让你的思绪平静下来，让自己感受到你的向导和保护者的存在。

(5) 站立，并面对相应的墙壁，当你召唤印章时，右手持权力刀放在胸前。刀尖应该是朝上的。一旦你感受到它们的存在，是时候点燃围绕圈的蜡烛了，从北方开始，向东，南方和西方。

(6) 在最后的祭品上写上Sahia，用你祭坛右边的蜡封上它。

(7) 敞开你的心灵，与这个恶魔相绑直到你完成或失败这个仪式。你可以尽情地命令这个恶魔，但要小心，一旦她被复活或者你在失败中释放她，她将迅速杀死你。

(8) 现在，你应该打开你的心灵，将她接纳到你的灵魂中，现在是时候与她合作，尽可能地通过第五元素摧毁。

(9) 允许剩余的能量返回它们的来源。

(10) 用athame以相反的方式，现在归还你剩下的能量，并按点燃的相反顺序熄灭蜡烛。

(11) 燃烧Azahran的觉醒。现在诅咒已经完成，你可以离开魔法阵了。等到第二天早晨清理材料并收起工具。不要烧掉最后的祭品，直到你复活Sahia并释放她到世界上。

咒语

(2) 听从我的声音！我呼唤守护者在光芒灿烂的白光中保护我。以元素的力量，我在这里将会安全。

(3) 大地！我请求你的力量！风！我请求你的智慧！火！我请求你的生命力！水！我请求你的坚韧不拔！

(4) 我接受这能量，并祈求它受到祝福，以保护我在通向无形界的旅程中。我召唤我的灵魂的力量和那些爱护并保护我的神灵，保护我的仪式场地，因为邪恶将在此召唤。根据神灵的意志和我的个人守护者，我请求我始终安全。

(5) 大地，北方墙的元素，封印。风，东方墙的元素，封印。火，南方墙的元素，封印。水，西方墙的元素，封印。精神，元素的捆绑力量，注意这里封印的印记。

(6) 现在是午夜，但天空之外是白昼，天堂之门将打开，力量将释放在大地上！我以我的名字（你的名字）的力量召唤不朽的星辰。五星，大开天堂之门，释放军团；如同肉紧贴骨骼，你贴着我的灵魂。

(7) 我命令你成为恐惧的极端形式，死亡，不可解释的死亡。作为未知死亡显现，因为你是真正的邪恶，死亡是你的玩物。成为毁灭一切凡人的厄运之手。

(8) 丢弃他们可怜的生命，我将成为死亡化身的预兆。你邪恶的恶臭将是他们腐烂尸体的气味。成为空气和厄运气息中的邪恶，所有必须死亡的人的厄运。

(9) 凭借我的权威之力，我打开这个圈子，并释放我的指挥守护者。

(10) 水！我释放你的力量离开我的圆圈！火！我释放你的力量离开我的圆圈！风！我释放你的力量离开我的圆圈！地！我释放你的力量离开我的圆圈！

(11) 向Sahia致敬，我的诅咒完成了。
