## 施法说明：'使某人不那么有吸引力'

**制作此法术所需物品如下：**

+   火柴（火柴棍）

+   污垢

+   蜡烛（小号，黑色）

+   图片

**制作此法术所需物品如下：**

+   火柴（火柴棍）

+   污垢

+   蜡烛（小号，黑色）

+   图片

*美丽学习的图片，

在**她**背后，这个法术在燃烧；

这肮脏，这污垢，图片制成，

美人的吸引力似乎在减退。*

当需要时，你可以用**his**替换**her**。
