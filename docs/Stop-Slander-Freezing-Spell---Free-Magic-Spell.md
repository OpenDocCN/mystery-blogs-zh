# 停止诽谤冻结咒语

这个法术的目的是阻止或停止对你在法庭或日常生活中的诽谤/流言。

## 施法指南：“停止诽谤冻结咒语”

**你需要准备以下物品来施展这个咒语：**

+   纸张

+   铅笔/墨水

+   杯子

+   酒精（水可以用作替代品）

+   意图

+   纸张

+   铅笔/墨水

+   杯子

+   意图

这是一个对付诽谤者或任何试图对你产生负面影响的人的好法术。如果你在法庭上有事，而你的对手在不诚实地试图影响他人，这个法术也可以使用。

这是一个冻结法术。

如果有人在个人或法律事务中对你不利，而你知道他们的名字，你可以尝试一种冻结法术。

使用一个新削尖的铅笔在羊皮纸上写下他们的名字。尽量让铅笔的尖端非常锋利。

将纸张对折，直到它无法再对折。

在你折叠的时候，注入你的愿望，让他们停止说谎、嘲笑或转向其他人。简言之，你试图冻结他们的行动（因此有了这个咒语的名字）。

然后将伏特加或变性酒精倒在纸上。建议在水槽上进行此操作。然后把它放在冰箱的角落里。

将它放在那里28-30天。有些人认为这应该在残月期间开始。

这应该可以制止诽谤。因此，它将他们冻结了。

随意添加任何咒语、蜡烛魔法等，将这个法术与你的意图融合在一起。记住所有魔法工作都必须充满你的意图。根据自己的舒适度调整事物。
