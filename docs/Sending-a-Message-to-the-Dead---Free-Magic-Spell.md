## '发送信息给死者'的施法说明

**你将需要以下物品进行这个咒语：**

+   1 根白蜡烛

+   1 根黑蜡烛

+   代表你希望发送消息的人物的物品（们）

**你将需要以下物品进行这个咒语：**

+   1 根白蜡烛

+   1 根黑蜡烛

+   代表你希望发送消息的人物的物品（们）

将白蜡烛放在左边（代表生命，代表你自己），在中间放置代表你希望沟通的人物的物品（代表你消息的对象，更具体地说，它指引你的消息到正确的灵魂。可以是代表他们的草药袋，水晶，照片等），右边是黑蜡烛（字面上代表已故的人）。

点燃白蜡烛，并通过首先召唤你信仰的神灵（沟通或与死者沟通能力的灵魂或实体）开启与灵魂沟通的方式。

告诉他们你希望通过消息来纪念这个人，请求将消息转达给他们或者是否允许他们聆听，并感谢灵魂/神灵。然后将黑蜡烛与白蜡烛一同点燃，并开始与他们交谈。当你完成传递消息时，祝他们一切安好并告别。

然后向促进沟通的神灵/灵魂宣告你已经完成，并再次感谢他们。接着转身吹灭黑蜡烛，说道：“当我吹灭这根蜡烛时，来自彼岸的沟通已经结束。任何不受欢迎的灵魂或实体都不能接触。”

然后继续吹灭白蜡烛，说：“当我吹灭这根蜡烛时，我结束了与神灵/神明/女神们的沟通。感谢你们的帮助和指导。”

在此之后，强烈建议对您进行过此类仪式的空间进行净化（以避免让自己和该空间对任何不受欢迎的灵魂开放）。
