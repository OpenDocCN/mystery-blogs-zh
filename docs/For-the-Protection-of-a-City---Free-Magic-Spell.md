# 为了城市的保护

一个保护城镇或城市的咒语。

**你将需要以下物品来执行此咒语：**

+   钱

+   面包（无酵）

+   橡树树皮

+   薰衣草

+   岩盐

## “为了保护城市”的铸造说明

首先，你必须保持八天的纯洁，从第一天日出开始到第八天日出结束。在此期间，你只能吃无酵面包和喝水。你不能进行性行为，也不能自慰。你不能偷窃，诽谤任何人，也不能伤害任何人。

在第八天，带上大量的钱，走遍你的城镇或城市。当你遇到无家可归的男人或女人时，给他们一些钱 - 足够吃一顿大餐。不要告诉他们你的名字，也不要使用化名，也不要告诉他们你住在哪里。完成后，离开他们，默默为他们的幸福祈祷；我不会告诉你要祈祷什么。

在日落之前至少重复八次，给至少八个赤贫者钱。完成后，返回家中。

在那里，将橡树树皮、薰衣草和岩盐一起压碎并点燃。完成时，说：“在你的圣盾下，我们都是安全的，哦，阿多纳伊，哦，耶稣·弥赛亚，无论是什么风暴或敌人，也不会触及我们。我们永远处于你的保护下。阿门。”

通过这个，操作将完成。没有瘟疫会触及你的城市，也不会有任何敌人毁掉它，也不会有可怕的风暴将其毁灭。

**额外说明**：

+   我将由从业者自行决定需要多少钱。我无法估计。

+   要慈善，但只给那些需要的人。许多人在等待，渴望拿走你的钱，而许多其他人确实需要帮助。不要给那些不需要的人，免得你引诱他们犯罪。要明智和观察，同时也要慷慨。如《教义》所说：“让你的赠与像汗水一样在你的手中，直到你辨明该给谁。”

+   非常重要的是，你不能告诉任何人你在执行这个咒语。不要告诉你的朋友、家人、魔法从业者同行，也不要告诉其他人。你还必须保持你的捐赠秘密，不告诉任何人你给了或计划给钱。“但你施舍的时候，不要让你的左手知道你右手所做的事，为要你的施舍在暗中行，你父在暗中看见，必要报答你。”（马太福音6:3-4）

+   盐象征着保存。

+   薰衣草象征着和平。

+   橡木与朱庇特、宙斯和托尔有关。

+   数字八与水星有关。
