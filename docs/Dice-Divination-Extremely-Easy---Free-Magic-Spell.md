# 骰子占卜极其简单

这是一个对我多次有效的咒语，每次以某种方式实现。（不要让任何其他人触摸你的骰子，否则它将不再有效。）

**进行此咒语所需物品如下：**

+   3个骰子，最好是新的，没有被其他人碰过的

+   1张方形羊皮纸/布/纸（我用纸巾）

+   1个写字用的东西，最好是记号笔

+   1个声音

+   2只手，最好是你自己的，附着在你的身体上（如果不是，愿你的灵魂受到祝福）

+   冥想的能力

## '骰子占卜极其简单'的施法说明

好的。首先，你需要准备咒语。

拿起羊皮纸或纸巾和记号笔。在每个角落大约3厘米处画一个圆圈，并拿起骰子。

我希望你静静地坐上一会儿，手中握着两只骰子。你必须把骰子放在手中，另一只手覆盖在上面。记住，冥想后你必须是唯一接触你的骰子的人。

我希望你继续坐在任何你觉得舒服的位置上，打开手掌看着骰子。对它们呼吸，同时想象光线从自然能量源（元素、精神动物或行星【我用月亮】是一些选择）进入你的头部，环绕你的手臂并进入骰子中。再次合拢双手封存能量。在大约10秒钟后说：

'愿神与我同在我的意志

为未来的歌曲啼叫

也许他们会对这些普通的骰子低语

告诉我，我的未来是美好的还是不好的。'

完成这个动作后，你再也不必为另一个问题重复此操作，除非你感觉到力量减弱。

现在，将骰子在羊皮纸上方的两只手中摇动，同时大声询问或强烈思考一个问题，例如，“我会很快结婚吗？”或者甚至是某些八卦，比如，“我最喜欢的作者会很快出新书吗？”其他人也可以让你提问，但他们不能触摸骰子。

无论哪个骰子落在圆圈内，你都必须注意。检查它们是否稍微触及轮廓——如果是这样，请不要计算它。

现在把圈内的数字加在一起。例如，如果圈内的两个数字是5，外面的数字是6，答案是10，因为5+5=10。

下面是数字的含义（1和2意味着你必须再试一次）：

3= 好消息，一个公告，或者一份礼物

4= 坏消息，麻烦，或者道路上的问题

5= 欢欣； 一个愿望成功实现

6= 悲伤，通常由不幸或失败引起

7= 挫折，通常由财务困难或流言引起

8= 可能升级为争论的误传

9= 人与人之间的和谐

10= 机会出现（可能是一个“是”，因为这是最好的

当问一个是或否的问题时)

11= 疏远，降低您对某人的喜好

12= 揭示，可能由媒体或朋友披露

13= 忧郁，一种不祥的感觉，您应该注意

14= 新朋友

15= 有人并非完全诚实…

16= 冒险，可能是出国旅行

17= 从改变您的战术和方法中取得成功

18= 非常积极的预兆，好运和大量某物

（可能是金钱）

希望这对你有所帮助。
