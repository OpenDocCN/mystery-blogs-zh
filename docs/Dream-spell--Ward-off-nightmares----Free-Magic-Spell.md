# 梦境咒语（驱散噩梦）

如果正确操作，会带来美好的梦境。咒语施错不会有任何效果。只有当你做噩梦时才这样做。

## '梦境咒语（驱散噩梦）'的施法说明

**你需要以下物品来进行这个咒语：**

+   一张纸

+   铅笔（施法时不能折断）

+   1 根白蜡烛，无香味。

+   （如果不可用）香味浓郁的蜡烛（白色）。

+   一个打火机

**你需要以下物品来进行这个咒语：**

+   一张纸

+   铅笔（施法时不能折断）

+   1 根白蜡烛，无香味。

+   （如果不可用）香味浓郁的蜡烛（白色）。

+   一个打火机

[必须在盈月期间进行]

1\. 用一支铅笔（必须是削尖的）在纸的中央画一个小云朵。然后写下一个秘密和一个任何时候的快乐记忆。折叠纸，并将其藏在除你以外的任何人都找不到的地方。（包括每个人，甚至你的宠物(s)）。

2\. 拿一根白蜡烛放在你的云中央。（展开纸）。不要点燃。

3\. 吟唱以下内容：

我的梦境已经控制了我，

对我来说是可怕的梦境，

神明的光与天堂，你啊，

赐予我心愿，

我希望不再有悲伤，

请赐予我美好的梦境。

4\. 吟唱之后立即点燃蜡烛，并将第一步的纸烧掉。当它开始疯狂燃烧时，将其扔进防火罐里的水中。扔掉纸。

什么算是施法失误?:

1\. 纸张不见了，有人找到了纸张。任何与纸张被发现的地点有关的事情。

2\. 你的铅笔断了，或者蜡烛点燃得太早。

3\. 咒语在早上/残月时进行。

4\. 有人进入你使用咒语的房间。

-

副作用：

你可能会遇到略微失控的梦境（疯狂的梦境）。

注意：这个咒语是50-50，意味着第一次可能不会得到美好的梦境。
