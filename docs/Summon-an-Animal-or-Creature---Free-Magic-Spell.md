## '召唤动物或生物'的施法说明

在开始之前，请阅读以下内容。

描绘你的动物的外观。根据它的元素给你的动物上色，或者只是给你的动物上色，然后根据它的颜色找出它的元素。画出你和你的动物的侧影。动物会比你高/矮这么多。

写下你的身高。你旁边。把你画出来（可选）。根据你的身高画线到你的身体顶部。 （例如：五英尺。画五条相等的线。）现在继续制作动物的高度。动物的肩膀顶部是它的高度。不是头。划线标出你和你的动物的身高。

把纸折成汉堡样。这样你就看不到你的图画了。只需折叠一次，现在用铅笔在任意一侧写下它的信息。（名字，毛色，翅膀，尾巴，配饰，力量，行为，能力，元素，品种，眼睛，它吃什么等等。）

再次将纸折成汉堡样，这样你就看不到字了。现在写下并大声朗读：“（它的元素）（我的生物），通过（它的元素的）蛋来到我身边。我召唤新生的神话生物。成为它所愿。”（"成为它所愿"应该尽可能大声地写出。）

现在将其折叠成汉堡样式，这样你就看不到字了。在一侧做一个五角星。再次折叠。然后，每天晚上三天，在你入睡前，吟唱：“上帝/女神，请将我的梦中（生物）从蛋到结束来到现实世界。”接着，吟唱：“（它的元素）（我的生物），借由（它的元素的）蛋来到我身边。我召唤新生的神话生物。成为它所愿。”

吟唱后亲吻纸，并将其放在枕头下。当你入睡时，梦见你和你的动物相聚。如果你在中途停下来，只需简单地读出你所写的。你需要真正相信魔法，否则你将得不到蛋。

+   水=坐在泳池里（如果你有的话）/蓝蛋

+   地球=岩石蛋

+   火=燃烧的蛋/红蛋

+   空气/风=也许是隐形的，强风来了/白蛋

+   雪=在冬天来临时出现/白蓝色调

+   金属=金属蛋

+   银=银蛋

+   铜=铜蛋

+   自然=多彩（我不太确定）

+   阴影/黑暗=黑色蛋

+   鬼=相位蛋

+   闪电=闪电击中的火花

+   极光=多彩的蛋
