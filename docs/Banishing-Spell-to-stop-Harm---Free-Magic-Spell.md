# 用于阻止伤害的驱邪咒语

这将保护人们而不会给被捆绑的人造成任何伤害。

这不是一种黑暗的咒语，而是一种非常有效的保护性咒语。

**你将需要以下物品来施展这个咒语：**

+   1根黑色蜡烛或形象蜡烛（性别取决于你试图捆绑的对象）

+   钉子

+   一大块黑布

+   红丝带棉花

+   针和线

+   驱邪油

+   散烟草

+   一个能独立站立的小镜子

+   如果你能从这个人那里得到头发或指甲剪下的，或者照片，你可以在这个咒语中使用它。

## 用于‘阻止伤害的驱邪咒语’的施法说明

将毛毡对折，剪出你想要捆绑的人的大致形状。

使图像足够大，以便你在缝合后能够填充它。

缝合小人的各个部分，留下一个可以填充的孔。

用棉花和烟草填满它，如果你有这个人的头发或指甲剪下的，也加入其中。

一旦填满，请把开口缝合起来。如果你有这个人的照片，

将其订在小人的正面上，用订书机或缝纫机。

接下来，用钉子在黑蜡烛上刻下这个人的名字，并添加这些符文：

Thuraz、Isa、Eihwas，一个填充的黑圆圈来代表黑月亮。

像你在监狱里看到的那样，和逆时针（逆时针）的螺旋。

用油抹润蜡烛和小人。

圈定一个圆，祈求元素，神或女神你正在和他们合作。

点燃祭坛上的蜡烛。

点燃黑蜡烛，并调整镜子，以便火焰反射在玻璃中。

把小人举在你面前说：

''布制品，你是，

你这个有血有肉的生物。

我宣布你（你要捆绑的人名字）。

不再伤害我。

不再编织虚假的故事。

不再干涉我的生活，也不再干涉我所爱的人的生活。

借助众神的力量和我的意志，如此成就吧！''

在小人上画一个召唤五角星。

现在拿起丝带开始像裹尸布般包裹小人，

一点空隙也不留。说：

''我束缚你的脚，使其不再伤害我。

我束缚你的手，使其不再伸向我造成伤害。

我束缚你的口，使其不再传播虚假的故事来伤害我。

我束缚你的思想，使其不再向我发送伤害的能量。

如果你继续这样做，请让所有负面能量直接反射在你身上！''

系紧丝带，把小人拿到镜子前，同时想象所有负面的

这个人送给你的能量被反射回他们身上。

用黑布包裹小人，用另一根丝带系紧。说：

''大地母亲，我已经将这个人束缚

阻止伤害我和我所爱的人。

通过三次三的力量

通过大地和火焰、空气和海洋

我解除这个咒语，然后释放它

它无害地返回给我

按照我的意愿，就这样吧！''

让蜡烛烧尽，而人偶坐在其底部，然后拿起人偶

将蜡烛残余物远离家并深埋地下

或将其扔进海洋，然后走开不回头看。
