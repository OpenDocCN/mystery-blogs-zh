# 看看你是什么超自然力量

这个咒语是一种力量咒语，可以解锁施法者的某些（而非所有）能力。如果他们有的话。这可能需要一周、两周，甚至一个月的时间，所以需要耐心！

**你将需要以下物品来施展这个咒语：**

+   罐子

+   蜂蜜

+   肉桂粉

+   柠檬或柠檬

+   花瓣 - 越大越好

+   纸

+   笔/马克笔 - 不推荐使用铅笔

+   雨水 - 或普通水

+   硬币

+   一些可以抽血的东西

+   线 - 可选的

## "查看你是什么超自然力量"的施法说明

这个咒语是新制作的，因此我建议用你自己的能量来充电，让它更有效。否则，这个咒语可能毫无用处。

**咒语**

冥想大约十分钟到一个小时。你可以随意冥想多长时间。只需确保专注并集中精力在你的能量上。

把罐子放在你面前，盘腿坐着（或者你感觉舒适的方式）。慢慢地向罐子里倒水。最好使用雨水。如果你不想等到下雨，可以做一个求雨的咒语。如果你想要一个真正有效的求雨咒语，就给我发邮件。

你只需要足够填满罐子底部的水。如果你不想等雨水，可以用普通水，但我不建议这样做。当你慢慢向罐子倒水时，念以下内容三遍：

"洁净我全身，

深深洁净我，

接受我的能量，

三次三次。"

现在，切一片柠檬或柠檬，放在一边备用。同时加一滴蜂蜜，撒一点肉桂粉，加半个或整个柠檬或柠檬，念以下内容两遍：

"地球生长的草药，

一锅女巫的魔药，

你的能量使我的愿望成真。"

现在，加上花瓣，然后再念一次上面的短语。现在，把嘴对着罐子，念以下内容三遍：

"生命的呼吸，灵魂的呼吸，

说出真相，挖一个洞，

埋藏谎言，

当它消失时，

把它藏起来，

今天不见了。"

拿一张纸，写下以下内容：

1.  你的名字

1.  出生日期

1.  我是什么样的超自然力量？

1.  当前日期

在纸上滴一滴血，念三遍：

"灼热的火焰，深深燃烧，

照亮路径，我将寻找。"

翻转纸。把硬币放在纸的中间，然后把纸折叠在硬币上。你写的字和血应该朝外。你可以用线绑在一起以防它散开，不过这是可选的。如果硬币和纸分开了，不要担心。

当你把硬币和纸一起放入罐子时，念三遍：

"许愿井，你能告诉我，

我的灵魂如此强大，

我的灵魂如此深，

展示给我真相，

三次三次。"

把罐子放在一个黑暗的地方，例如你的床底下。如果你开始运气不好，立即冲洗罐子及其内容。

如果你开始感觉更强大，更有能量，或者开始注意到能力，请让罐子待着。一旦力量和能量消退，或者当你不希望法术继续起作用时，请冲洗罐子及其内容。祝你好运。
