# 已知与未知的治疗

这里有两个不同但相似的咒语，用于治愈你认识的人和治愈你不认识的人，比如朋友的朋友。

## "治疗已知与未知"的施法说明

**你将需要以下物品来进行这个咒语：**

+   某物象征五大元素（土、气、火、水和精神）

+   4支白色或蓝色蜡烛（健康）

+   黑色或红色蜡烛（伤害）

+   某物来代表这个人

+   一个碗（任何种类）

**你将需要以下物品来进行这个咒语：**

+   某物象征五大元素（土、气、火、水和精神）

+   4支白色或蓝色蜡烛（健康）

+   黑色或红色蜡烛（伤害）

+   某物来代表这个人

+   一个碗（任何种类）

好的，所以我写了这个咒语，请不要窃取它，我是从我的魔法书中得来的，所以我们开始吧。第一个是治愈你认识的人。

说：

"愿他们被治愈

由生命之元素

大地、气息、火焰、水。精神充满

流过并围绕它们

治愈他们的痛苦和

保护他们免受死亡的伤害。"

当你说出元素时，将代表每个元素的东西放入碗中，并与代表你治愈对象的东西一起放入。在咒语开始时，点燃黑色或红色蜡烛（颜色不重要）。在咒语结束时，吹灭点燃的蜡烛并点燃白色或蓝色蜡烛（同样，颜色不重要），这代表痛苦离去，健康取而代之。拿起健康的蜡烛并将碗中的东西燃烧。

现在是治疗你不认识的人的咒语。说：

"让（在这里说出人名）从痛苦和死亡中得到治愈

愿精神和大地成为一体

并治愈这个人的心灵、身体和灵魂

让他们过上充实的生活"

当你说精神和大地向紫水晶撒盐时。在咒语结束时点燃一支白色或蓝色蜡烛（任何一种都可以）。重复这个过程3次，每次都要重复这些步骤。
