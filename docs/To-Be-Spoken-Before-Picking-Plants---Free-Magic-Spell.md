# 采摘植物前的祈祷

这是一种在采摘任何用于魔法的植物之前要诵读的咒语。改编自希腊魔法纸草卷中的一个咒语。

## 施法说明：“采摘植物前的祈祷”

**你将需要以下物品来完成这个咒语：****你将需要以下物品来完成这个咒语：**在日出前的任何与特定草药相关的日子（例如：草莓为星期五，五叶草为星期三，土星为土豆等）诵读此咒语：

“我用五指的手采摘你，PN。我，NN，将你带回家，让你为我服务，为了某个特定的目的。我以神明DN的无瑕之名起誓，如果你不听我的，如果我的操作失败，那么孕育你的大地将永远不再被浇灌。Mutabar nak barnakoka braeo menda laubraasse bendeo。为我实现完美的魅力。”

在“PN”处，说出植物的名字。在“NN”处，说出你自己的名字，无论是你的出生名字还是一个魔法名字。在“DN”处，说出与植物相关的希腊神明的名字（爱神阿芙罗狄蒂代表金星植物，墨丘利神赫尔墨斯代表水星植物，时神克洛诺斯代表土星植物，宙斯代表木星植物，战神阿瑞斯代表火星植物，太阳神阿波罗代表太阳植物，月亮女神阿尔忒弥斯代表月亮植物）。

**附加说明**：

+   在采摘植物时，最好面向东方，因为那里是太阳升起的地方。

+   你必须用手采摘植物；使用剪刀等工具会削弱咒语的效果。

+   大多数植物不属于单一行星。如果你想要的植物属于多于一个行星，使用与你意图最相关的希腊神明的名字。例如，如果它既属于木星又属于金星，使用爱神阿芙罗狄蒂代表爱情，或使用朱庇特代表财富或名望。
