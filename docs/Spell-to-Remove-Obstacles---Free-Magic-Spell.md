# 消除障碍咒语

障碍会阻碍和束缚你的生活。如果你曾遇到过无法前进的阻碍，很可能是由于生活中一些未解决的问题。努力追求宽恕、目标、接受、责任，都是困难的任务。我们需要记住的是，虽然我们对影响我们的外部事物或其他人对我们所做的事情可能没有责任...但我们对如何应对这种情况负有责任。这是一种强大的咒语，可以对抗这些未解决的问题，但你必须准备好前进和继续前行。这个咒语...关乎释放。你可能会哭泣，你可能会大声喊叫，但你也将愈合、释放和前进。在满月或残月期间进行这个咒语。

**进行此咒语需要以下物品：**

+   酸橙一半

+   柠檬一半

+   橙子一半

+   淋浴（或瀑布）

+   一支黑蜡烛

+   一根别针或针

## '消除障碍咒语'施法说明

点燃黑蜡烛，沉思火焰。在冥想火焰时，将针或别针穿过。用刚加热的针或别针在蜡烛上刻下“去除障碍”字样。想象自己快乐健康，看到障碍被解决，当你宽恕，愈合，继续前行并对自己的生活负全责时。蜡烛燃烧时，带着水果的一半进入淋浴间。从橙子开始，温和地将它在身上移动，就像在用橙子清洁身体一样。

朝远离你身体的方向移动。从头顶开始，朝着脚底移动。

橙子：在处理橙子时，想象曾经因你的刻薄、愤怒、粗鲁而受到伤害的人。感受内疚，并允许自己哭泣。当你准备继续前进时，说出以下话语；“对不起，请原谅我，我爱你，谢谢你。”这是Joe Vitale的一种强大的肯定。

柠檬：想象曾经伤害过你的人。他们可能是故意伤害你的，也可能是无意间伤害了你，但这仍然让你受伤。允许自己感受这种伤害。是的！这并不好，但它发生了。当你准备真正放下这段过去，向前迈进时，说出以下话语；“你伤害了我，但我不会让这种痛苦掌控我的生活。我原谅你”（让自己感受内心平静，并真正感受到你已经原谅了他们）。

酸橙：想想你自己。我们对自己的惩罚比对任何其他人都更多。我们对自己是如此刻薄和苛刻，但这是我们的生活，你足够漂亮。你足够聪明。你如何对待自己？为什么你不爱自己？为什么你伤害自己？放手。决心改变你的生活，成为你一直想成为的人。因为你体重超标而对自己没自信？决心做得更好。决心在所有事情上做得更好。拥抱自己。与自己合而为一，当你准备好时说：“我原谅自己过去的任何过失或短处。我决心对自己的生活负百分之百的责任，我准备好过得快乐。我接受幸福。”

熄灭黑蜡烛并远离你的家。完成后，你会感到虚弱和情感枯竭。此过程结束后，你可能想要睡觉。如果你仍然在与过去的痛苦作斗争，你可以根据需要重复这个过程。

**警告** 你将独自离开蜡烛去洗澡。把蜡烛放在一个安全的地方，比如壁炉上，蜡烛台上等等……
