# 星辰阿尔凯德的咒语

一篇祷告给星辰阿尔凯德（大熊座主星），以使某人倒霉。

**进行此咒语所需的物品如下：**

+   草药：安息香、沉香、芳香根、葡萄酒、鹿尾骨。

+   设备：一个装有热炭的香炉。

+   袍服：一件白色长袍，配有蓝色兜帽。

星座：

当星辰阿尔凯德处于第十宫时，夜间出行。尽可能地将运势带到第十宫和上升星座，并让上升星座的主宰和第十宫的主宰互相友好；让它们远离一切不幸，并且在它们的尊严中处于良好的地位，并相互显现。

熏烟：

将安息香、沉香和芳香根各两盎司混合，鹿尾骨捣碎一盎司，与上等老酒混合，并将混合物制成片剂，以备后用。

咒语：

将一些药丸放在香炉中的热炭上。烟雾上升时，面向阿尔凯德所在的天空部分，并说：

"有害者你变成了，本尼斯。熊将你从她摇摆的尾巴上甩下来，她的怒火降临在你身上。自豪地前行，本尼斯！在你的愤怒中口渴地扼杀敌人。"

然后将自己摔倒在地上，多次说出并做这些事情，呼唤你希望降祸的人并祈求并呼唤星辰。
