# 火球-基于火的咒语

这个咒语使你能够创造一个火球。它需要大量的能量，但经过大量的练习后，你可以无需浪费一口气就创造出几个！有许多不同的版本，其中许多更强大，但对初学者来说，这是最好的。

## '火球-基于火的咒语'的施法说明

**你将需要以下物品来完成这个咒语：**

+   能量（非常重要！）

+   一个朋友

+   有关魔法火的经验

**你将需要以下物品来完成这个咒语：**

+   能量（非常重要！）

+   一个朋友

+   有关魔法火的经验

（我在配料中包含魔法火的原因是因为它与通常用于攻击和防御而不是保暖不同。如果你希望保暖，请点击我的历史框查看我第一个关于保暖的咒语）

1\. 坐或站在你朋友的身边。在练习期间有一个朋友总是好的，以防你所有的能量都被耗尽。

2\. 接下来，专注于你的手。设想它，以便在你的眼皮下记住每一条皱纹和每一根毛。 （这部分在许多月的练习后并不总是必要的。）

3\. 聚集在你手指尖的温暖，并低声念以下话语：

让火温暖我的手

以及对手地上的冰的影响

让他的灵魂在悲伤中燃烧

当他像叶子一样枯萎时

4\. 上述部分可以大声说出来，在你的脑海中，或在你的头脑中设想：最常见的选择是设想这些词语，因为这样更快。

5\. 这就是你的朋友派上用场的时候。所有的能量和血液的注意力都集中在建筑热量上，很快就会成形。能量和信念越多，效果越好。如果不起作用，不要惊慌。只需多重复几次诗句。如果你是一个非常初学者，我建议你要有耐心。我仍然没有掌握这个咒语，但经过几个月的练习，我已经创造了一两个。

6\. （可选）要创建许多个，你只需设想同样多的手。

如果你想要六个球，你有六只手，每一只都是另一只的完全复制。这需要不少时间，但需要大量的能量。

7\. 请保持安全并享受乐趣！

8\. 最后，要把它拿走，你要用力吹，除非你已经用过它，例如在决斗中。
