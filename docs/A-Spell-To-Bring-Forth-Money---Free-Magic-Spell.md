# 一个招财咒语

评分 5/5 星

这个咒语会给你钱，同时也会驱散厄运。所以基本上是一个两全其美的咒语。

**你将需要以下物品进行这个咒语：**

+   一滴蜂蜜

+   你的一根睫毛

+   你的血（很少）

+   硬币

+   纸，笔

+   水（建议雨水）

+   线（任何颜色，但白色或黑色更好）

+   熏衣草喷雾（或熏衣草叶）

+   绿色蜡烛

+   蓝色蜡烛

+   各种蜡烛

+   叶子（可选）

+   两个罐子/杯子

+   一个盖子，或者可以用作盖子的东西

+   在内部，独自一人

## 施法说明书：“一个招财咒语”

冥想约十分钟。这是可选的，但建议这样做。

现在，坐在地板上（里面）。把两个罐子或杯子放在你面前。把蓝色和绿色的蜡烛放在罐子的中间，其他蜡烛放在罐子后面或围绕着它。

把以下物品按顺序放入左边的第一个罐子里；

-加入雨水（或水），让其覆盖杯子或罐子的底部。

-加一滴蜂蜜

-加入你的一根睫毛

-把硬币放进去（如果你有一枚25美分的硬币，那更好。）

-放下叶子（可选）

-现在，在一张小纸上（宽度和长度约为一英寸），写上“给我钱”的字样，另一面写上你想要的金额（可选）。尽量保持较小的金额。不要写上一百万美元或者其他很大的金额，否则不会起作用。金额还取决于你的年龄和是否有工作。如果你是一个没有工作的孩子或青少年，可以写上你的爸爸妈妈可能会给你的金额（比如，20美元或100美元），用于去商场等。

-在纸上滴上一滴你的血

-把纸放进罐子里。

把嘴对准罐子但不要吸入，然后重复念三次;

''钱，钱，来吧，

给我想要看到的东西，

净化，

能量，

为了所有的善良，

这是我的意愿，

愿如此''

现在，把罐子放回原处，并盖上盖子。如果你没有盖子，可以用保鲜膜或锡纸覆盖，然后用发带、橡皮筋或绳子封口。

现在，用足够的雨水（或水）装满右边的第二个罐子，以覆盖底部。

把熏衣草叶放进去，或者喷三次熏衣草喷雾。

取一张小纸（宽度1英寸 x 长度），用血画出Gibu Auja符号。如果你不知道那个符号是什么样子，这里有一个链接。如果链接打不开，可以谷歌一下。

[http://postimg.org/image/9faf84997/bb255cbd/](http://postimg.org/image/9faf84997/bb255cbd/)

现在，把纸放进罐子里，然后把嘴对准罐子但不要吸入，然后重复念三次;

''幸运，

保护，

来吧，

这是我的意愿，

愿如此''

把罐子放回原处。如果你没有盖子，可以用锡纸或保鲜膜。

现在，专注于你的能量，蜡烛的火焰和你面前的两个罐子／杯子。想象结果，（这可能是你在工作时领到工资，得到加薪，或者你的爸妈给你钱）

然后，吹灭蜡烛。

把罐子放在一个黑暗、隐蔽的位置。（壁橱，柜子，床底下等）

得到你的钱后，或者如果开始遭遇厄运或任何负面后果，立即冲洗罐子，丢掉纸条。
