# 完成驱邪仪式

评分5/5星

这个咒语是一个完整的防御负能量的逐步仪式。在盈月时做最好。

**您需要以下物品来完成此法术：**

+   盐（足够做一个你可以坐在里面的圆圈），

+   4支黑色或白色蜡烛（代表北、东、南和西四个方位），

+   代表四大元素的物品（土 - 来自花园的泥土，空气 - 一根羽毛，火 - 一支黑蜡烛，水 - 一只玻璃碗盛水），

+   一个标识符（你的一滴血或一缕头发），

+   代表你想要驱邪的人负能量的东西（任何对你来说带有不良气息的事物），

+   一大段黑色绳子，

+   一把权杖（从未使用过的黑色把手的刀子），

+   一个五角星（圆圈中的五角星）。

+   一个健康的小吃（水果块），一些饮料（葡萄酒被认为是纯净的），

+   一个耐火碗，

+   火柴，

+   山艾

+   迷迭香。

如果你确实需要离开圆圈，你必须首先切开一个“门道”。
