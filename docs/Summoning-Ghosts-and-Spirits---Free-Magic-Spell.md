# 召唤鬼魂和精神

评分为5/5颗星

这是一个非常简单的法术。你可以随时使用它。

## 为'召唤鬼魂和精神'施法说明

**你需要以下物品来进行此法术：**

+   一张纸。

+   +1 把缝纫针

+   +羊毛（红色）。

+   +5 蜡烛。（颜色无关紧要）

**你需要以下物品来进行此法术：**

+   一张纸。

+   +1 把缝纫针

+   +羊毛（红色）。

+   +5 蜡烛。（颜色无关紧要）

从你的笔记本上撕下一张纸。在中间做一个更大的斑点（它应该是红色的，用彩色铅笔做）。然后拿起羊毛并把它放在针的顶部孔里。这是一个缝纫针，让它很小。

在中间的点附近写下这些词：“YES”，“NO”，“MAYBE”

和“我不知道”。它们应该间隔较远。

仪式：

你把针放在红点上并用羊毛固定。不要握住针，只握住羊毛。现在你念：

“我呼唤__________的鬼魂/精神”（你想召唤的那个）

你可以召唤你想要的任何人。（例如，如果你有一个已故的表兄弟或家庭成员，你可以召唤他/她的灵魂，并询问他/她任何你想要知道的事情）你会三次念咒语。当你说这些话时，记得用手转动针（只用一只手）

现在你可以问一个问题，并等待答案。尽量不要移动针。当针靠近答案时，你就会知道答案（从你写的答案中）
