## 哮喘的症状和迹象

哮喘的症状可能很轻微，常见的症状包括：

+   慢性咳嗽，喘息

+   紧张或胸痛

+   运动时呼吸急促

+   由于喘息、咳嗽和呼吸困难而导致的睡眠困难

### 维生素E和维生素C食品
