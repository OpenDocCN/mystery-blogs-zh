# 成为淡水或盐水人鱼

这将教会你如何成为一直想成为的人鱼。

## 施法说明：'成为淡水或盐水人鱼'

**施法所需材料：**

+   1 925银项链/手链

+   1只玻璃杯

+   一碗清水或盐水

+   1只白色蜡烛放在银烛台上

**施法所需材料：**

+   1 925银项链/手链

+   1只玻璃杯

+   一碗清水或盐水

+   1只白色蜡烛放在银烛台上

点燃蜡烛。拿起玻璃水碗，将银项链或手链浸入其中。确保它被浸湿。拿起手链或项链，悬在火上。在火上方，念咒语：

"海洋姐妹们来到我身边

带来海洋的力量

让我们与海洋和水融为一体

我不是鱼也不是人"

将项链/手链放入火中等待6秒。将碗中的水倒在蜡烛上，并戴上项链。

项链/手链是一个解除咒语的护符。如果你想在水中游泳而不成为人鱼，那么在水中戴上项链/手链。在水外的所有时间都要戴着，否则它会失效。
