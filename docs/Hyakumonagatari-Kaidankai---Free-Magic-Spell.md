# 百物语怪談会

评分为5/5星

与我们迄今为止介绍过的许多游戏不同，日本的仪式百物语怪谈会（更准确地翻译为“百个超自然故事的聚会”）有着悠久的历史。最早记载的版本可以追溯到1660年；最初被认为是武士的勇气考验，尽管随着时间的推移，它在社会中流传并演变为今天所玩的版本。请注意，这个仪式打开了一扇进入灵界的窗户（可能是阴影界），但并未指定如何在打开窗户后关闭它。

**完成此咒语需要以下物品：**

+   100支蜡烛。

+   点燃火柴或打火机。

+   一个黑暗的房间。

+   至少一个讲故事的人，或者多达百人。

## “百物语怪谈会”的施法说明

前奏：

1.把100支蜡烛放在指定房间的中心。

2.关闭所有窗帘并熄灭所有灯光。现在外部光线应该渗入房间；必须实现绝对黑暗。

3.等待夜幕降临。

主要事件：

1.在日落后开始。

2.聚集所有讲故事的人在一起，进入房间。围绕蜡烛形成一个圆圈。

3.点燃全部100支蜡烛。

4.轮流讲述奇怪和不寻常的故事。这些故事不必是真实的，尽管它们可能是；它们也不必发生在讲故事的人身上，尽管同样地，它们可能发生过。

5.每讲完一则故事，讲故事的人必须吹灭一支蜡烛。

结束：

1.当最后一支蜡烛被吹灭时，做好准备。窗户将被打开。接下来会通过窗户进来什么，任何人都无法预料。

变体：

这个仪式有多种变体。以下是一些选择。

1660年版本：

注意：这个变体最早记录在1660年安政荻田所著的怪谈集《十能草》中。

1.在新月开始。

2.准备三个房间。这些房间应相邻；从上面看，它们应该形成一个“L”形。每个房间内都要清除任何武器或危险物品。在第三个房间中，放置100盏由蓝色纸制成的行灯和一张桌子；并在桌子上放置一面镜子。确保所有三个房间完全黑暗。

3.聚集所有讲故事的人，进入第一个房间。讲故事的人应穿着蓝色服装且未携带武器。

4.轮流讲述奇怪和不寻常的故事。每讲完一则故事后，讲故事的人必须在黑暗中从第一个房间走到第三个房间。到达第三个房间后，讲故事的人必须吹灭一盏行灯并在镜子中看自己。只有这样，讲故事的人才能回到第一个房间。

5. 讲故事者可以在上一个讲故事者旅行时继续进行仪式。

6. 当最后一个行者熄灭时，真正的黑暗将降临，可能会召唤出灵魂。

十个故事的游戏版本（Jumonogatari）：

注意：这个变体只涉及一个讲故事者和十个故事。它不需要在一个黑暗的房间中进行，并且可以在几天的时间内完成。

1. 点亮10支蜡烛。

2. 开始讲述奇怪和不寻常的故事。

3. 每个故事结束时，讲故事者必须吹灭一支蜡烛。

4. 当第十支蜡烛熄灭时，讲故事者将被允许一瞥冥界，并与其中所居住的任何事物交谈。
