# 赋予你力量的饼干

评为5/5星

吃起来不错，很适合在一个艰难的魔法之前/之后食用，使之更强大/更好。

## 施法说明'赋予你力量的饼干'

**您将需要以下物品来进行此法术：**

+   2又3/4杯普通面粉

+   1茶匙小苏打

+   1/2茶匙发酵粉

+   1杯软化的黄油

+   1又1/2杯白糖

+   1个鸡蛋

+   1茶匙香草精

+   4到5汤匙的酪乳

+   1汤匙咖啡

+   2颗压碎的茴香籽

**您将需要以下物品来进行此法术：**

+   2又3/4杯普通面粉

+   1茶匙小苏打

+   1/2茶匙发酵粉

+   1杯软化的黄油

+   1又1/2杯白糖

+   1个鸡蛋

+   1茶匙香草精

+   4到5汤匙的酪乳

+   1汤匙咖啡

+   2粒压碎的茴香籽

将烤箱预热至375华氏度。

在一个小碗中，搅拌在一起面粉，小苏打，咖啡，发酵粉。搁置一边。

在一个大碗中，把黄油和糖搅拌到光滑。加入鸡蛋和香草。逐渐混合干成分。加入足够的酪乳使面团变软而不是湿润。

把面团捏成圆形，放在未涂油的曲奇烤盘上。用刷子或手指，将每个饼干的顶部涂湿，然后略微压平每个饼干的顶部。用原糖或彩色糖珠洒一些。

烤8到10分钟，或者直到略微变金色。取出晾凉前静置2分钟。
