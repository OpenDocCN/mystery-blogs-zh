## 使用你的石头神谕：布局或投掷

现在你又有一个决定要做。你打算如何使用你的水晶，是像塔罗牌一样摆放它们，还是像占卜符文一样投掷它们。布局有两个缺点。首先，当你摆出水晶时，你可能会下意识地通过它们的形状来识别石头。这种识别会影响到随机性，并且可能会偏离读取结果。请记住这一点，或许努力获取形状和大小非常相似的石头。其次，如果你有圆形的石头，它们不像塔罗牌那样有“颠倒”的含义。不过，需要注意的是，有些读者只使用塔罗牌的正位解释，因此这可能不会影响你的努力。

投掷要简单得多。首先让求问者挑选一块代表他/她自己的石头。将其放在你的投掷布的中心点。然后把石头给求问者，让他们专注于一个问题或情况。最后，当他们觉得石头已经“热身”时，他们将它们洒在表面上。如果一块石头落在布料边界之外，它不计入你的阅读结果。

现在你可以更仔细地观察每块石头落在哪里。观察水晶是否：

+   彼此靠近。这通常涉及私密事务或对求问者个人影响深刻的事物。

+   成行排列。这代表线性思维或在解决方案方面把自己限制起来。

+   指向求问者或另一块石头。把它想象成箭头。那就是这种特定水晶能量的移动方向。

+   离开“自我石”很远：可能会无意中影响到情况的边缘事物（或人物）。

+   以特定的模式。方形代表基础和稳定，圆形代表平等，上升的三角形代表我们的三重本质，等等。

这些观察可以增加水晶本身的含义。现在你可能会觉得这种方法对你来说太随机了。如果你想寻找一种扩展你的阅读能力的方法，你可以创建一个个性化的投掷布。

**设计你的投掷布**

取一块布，你可以很容易地在上面涂上油漆或标记。一个12×12英寸的布料效果还不错。你甚至可以找到一块餐巾纸来完成这个任务！

接下来，考虑一下你希望在表面上布置意义的方式。你会做一个符号或文字的圆圈吗？每个角落都会有一个指定吗？以下是一些可能有所帮助的想法：

+   在布料的四个角落标记四个方向。

+   在这四个点之间加入季节：东北（冬季）、东南（春季）、西南（夏季）和西北（秋季）。

+   将月相放入，可以使用传统符号/标题，也可以使用由美洲原住民在全年中分配的满月名称。

+   包括元素。这些可能标记在四个方向或四个季节的内部或外部。

+   在布料上有一个“自我”点（通常位于中心）。

你可以在你的布料上使用其他潜在的符号，如树木、花朵、动物等的魔法本质。只需记住，你想要的符号越多，为了清晰起见，布料就越大。

**标记布料**

一旦你有了你选择的符号清单，你可以尝试一个样本布局。为你的每个符号或词使用小纸片，每个一个。将它们放在你打算放置它们的位置上。看起来如何？感觉对吗？如果不对，继续移动你选择的符号和词，直到你对配置感到满意。将其视为为你的占卜工具进行风水布局！
