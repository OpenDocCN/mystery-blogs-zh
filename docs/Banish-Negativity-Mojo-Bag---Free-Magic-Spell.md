**你需要以下物品：**

+   小黑袋

+   熏香棒

+   黑色蜡烛

+   海盐

+   香芹

**对于此咒语，你需要以下物品：**

+   小黑袋

+   熏香棒

+   黑色蜡烛

+   海盐

+   香芹

+   迷迭香

+   薄荷

+   丁香

+   海螂贝

+   玛瑙

+   没药熏香

+   建议使用轭香油

在你施法的地方收集物品。点燃熏香和黑色蜡烛。

用黑蜡烛点燃熏香。如果你喜欢，可以用烟圈圈绕。

一手拿着黑色袋子，一手拿着熏香，以熏香围绕袋子三次。让袋子吸收熏香的烟雾和蜡烛的一些热量。小心！别太接近。用贝壳和石榴石重复这个过程。

将草药放入袋子中，连同贝壳和石榴石。封口后涂抹袋子。用你的能量滋养袋子。完成后，说：

**"我赋予这些物品力量，

黑暗将畏惧，

用这个袋子来排斥负能量和负面的想法，

用这个咒语全部驱除。"** 

将袋子放在熏香附近直至熄灭。你可以把袋子放在任何你喜欢的地方。所有物品都是可选的，你可以进行替换。
