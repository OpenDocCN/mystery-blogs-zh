# 吸血鬼五角星仪式

评分为5/5颗星

吸血鬼驱逐五角星仪式（VBRP）：用于将心灵吸血鬼从你的魔法工作中驱逐出去。基于经过验证的黄金黎明赫密特魔法。

**你将需要以下物品进行这个咒语：**

+   木桩（或银祭刀）

+   能够想象出火焰形状

+   仪式空间

## '吸血鬼五角星仪式'施法说明

1) 面向东方开始并画一个圆圈。你可以用木桩指向并将其想象成在地板上环绕仪式区域顺时针行走时的火环。

2) 用你的支配手（或拇指旁边的前两根手指）触摸列出的身体部位，说出旁边的希伯来语单词，执行卡巴拉十字架。

额头：阿特

胸部：马库特

右肩：维格布拉

左肩：维格杜拉

交叉双臂，手放在肩膀上说：**永远阿门**

3) 现在我们刻画四个地球五角星。换句话说，保持手臂向前伸展，描绘和想象在基本方向上有四个大的火焰五角星。（顶部有一个点）正确刻画地球驱邪五角星的方法是，从左下角开始，然后向上，然后到右下角，然后到左上角，直接到右上角，然后回到左下角。

面向指定方向，同时在地上画五角星的过程中发出希伯来神的名字。（再次顺时针旋转）（YHVH，ADNI，AHIH，AGLA）如果你不喜欢希伯来神，你可以说出其他实体的名字。此外，五角星本身并没有任何撒旦主义的含义，因为它们在撒旦主义出现很久之前就被用于魔法中。

东方：耶和华

南方：阿多奈

西方：埃赫耶

北方：阿格拉

手仍然向前，返回东方。

4) 这是我们通常祈求大天使的时刻。交叉双臂放在身前，肘部朝前，说出以下内容：

> **"在我面前，拉斐尔
> 
> 在我后面，加百列
> 
> 在我右手边，迈克尔
> 
> 在我左手边，奥利尔。**

**注意**：如果你对恶魔感兴趣，你可以在这里召唤地狱之王而不是大天使。（路西法，利维坦，弗勒乌斯，贝利亚尔）灵体的名字应该被颤动，并且应该尽力想象这些灵体。

**说**：

> **"在我周围燃烧着五角星，在柱子中闪耀着六射线星。我从邪灵和恶魔中驱逐出去。"**

5) 执行卡巴拉十字架。（步骤2）再次跺右脚。
