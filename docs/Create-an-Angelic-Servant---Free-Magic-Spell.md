这是如何制造一个属灵的天使仆人来履行你的命令。非常适合发送祝福或进行治疗。

## 为'创造一个天使仆人'的施法说明

第一版本：使命天使仆人

这个版本的天使仆人将会被创造，执行你的意愿，然后与你的能量融合，或者返回到天使王国。

像往常一样进入冥想状态。确保自己做好净化和中心化。

当你准备好的时候，看到你头上的光。光无处不在地闪耀着，并像有人拿着一块大毯子高高举起。在其中是来自天使王国的无尽光芒。愈合天使的歌唱，感受它们在你的上方的存在。

现在使用以下任一种的想象力。第一种是想象一束光从天上射下，并在你面前形成一个光球。第二种是看到光在移动，一滴光落下并悬浮在你面前，就像一大滴雨滴。

这个光球是天使之光，是由源光和天使之光融合而成的。看到光如何移动 - 闪闪发光、脉动的光，像水波一样涟漪。尽量使球尽可能坚固和浓缩，并加入你需要的尽可能多的光。给你的仆人一个名字，或者问它想被叫什么。这个球有自己的意识，但可以由你来操纵。它没有能力做任何对所有人最有利的事情之外的事，因此告诉它，你是它的主人/指挥官，它是你的仆人，并且 - 如果你的请求对所有人的最高利益不利 - 它将使之更完美。

现在清楚地考虑你的意愿，详细说明你希望做的事情，以及其应该如何发生，还有按照什么顺序进行。尽可能清楚地想象这些事情，然后把这些指令以心灵力量发送到天使仆人那里。注意，一旦任务完成，它必须完全融入你的能量或返回到天使王国。

想想你如何能够在一小句话中迅速表达你想要的，然后使用那个句子下达指令。例如，''去发送___一大包源光，然后返回到我这里并完全融入我的能量。''或者，''去治疗___，然后返回到我这里并完全融入我的能量。''如果你愿意，可以尝试加上一个时间限制。例如，''去给___进行三天的治疗，然后返回到你来自的天使王国。''

在你完成指示后，心灵地将仆人投掷出去完成它的工作。

第二版：绑定的天使仆人

这个版本会创造一个天使仆人并将其绑定到一个物体上（通过链接或占有）。无论是谁穿戴/携带或者将该物体放在离他们三英尺范围内，天使仆人都会对他们起作用。

链接绑定：

进行冥想，创造一个天使仆人，然后对其进行编程以做你想要的事情。在这个例子中，我们将创建一个治愈的天使仆人。事先将你的物体放在你面前。看到一根能量绳从仆人流下，连接到你想要它绑定的物体上。绳子接触到物体，其能量完全充满它。注意绳子是仆人的延伸，像一只手臂，所以仆人通过它来充满物体。链接方法允许仆人移动，同时通过物体引导其能量。在加强绳子的同时，发送你允许仆人旅行多远的意向。你可以让它去任何地方，或者可能只是几英尺以内。非常清楚地表达并将意向发送到绳子中。清楚地陈述（心理上或口头上），比如，“仆人，你永远通过你的绳子被绑定在这个___上，直到释放。你可以去你想去的任何地方。”或者一个替代结束，“你永远不可以离开___超过三英尺。”

从这里，你会指示仆人做你想要的事情。所以我们发送了我们希望它给予的治愈种类的意向给任何附近或身上有这个物体的人。请明确，它只影响那些靠近、穿戴或携带该物体的人——没有其他人。同时要明确它是否一次只影响一个人，还是如果附近/触摸该物体的人很多，是否会影响更多人。

拥有绑定：

按照链接绑定的说明操作，但是不是创建一个链接，而是实际上看到仆人完全完全地进入物体内部。给它命令，“仆人，你被绑定在这个___里永远留在内部，直到释放。”

当你完成了给它的指示后，看到一层白光覆盖着物体，将仆人封印在其中。

如何从物体中释放天使仆人：

向仆人吟唱，“仆人，你现在被释放出来了___！”如果你采用了链接方法，请看到它的能量从物体中消失，绳子缓慢收缩，能量回到仆人的球体中。如果你采用了占有方法，请完全移除白色层（通过消退、流失、碎裂、抬升等），然后看到仆人的能量缓慢从物体中流出。如果它仍然保持物体的形状，没问题！

现在命令对象与你的能量完全融合，或者返回天使境界。不仅在心灵上，还要用语言表达。''仆人，你现在必须完全融入我的能量中''，或者''仆人，回到你来自的天使境界！''注意，你可能需要多次说这个。
