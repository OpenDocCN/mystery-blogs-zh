# 房间/场所的普通净化咒语

我不确定这是一个咒语还是一个仪式，但我用它来净化我的房间，在我的信念中，它有效果。

**你需要以下物品进行这个咒语：**

+   鼠尾草

+   一个能代表你的水晶

+   你喜欢的熏香（我用了茉莉花和洋甘菊）

+   一个镜子盘，或一个盘子

+   一根白色的蜡烛

+   一些月亮充能的水

+   一个装水的碗

+   一些盐

+   对北、南、东、西方向的了解（如果你不是在净化一个房间）
