# 繁荣沐浴与金钱仪式

今天的这个繁荣/金钱浴咒语有所不同-这是一个不错的传统咒语，其中所有组成部分都像应该的那样一起发挥作用，以创造一个能够实现意图的门户，即特定金额的钱。

## “繁荣沐浴与金钱仪式”的施法说明

**你将需要以下物品来施展这个咒语：**

+   1只绿色蜡烛

+   3枚银币

+   盐

**你将需要以下物品来施展这个咒语：**

+   1只绿色蜡烛

+   3枚银币

+   盐

拿起绿色蜡烛，在蜡烛上题写你需要的金额。 不要贪婪，要合理。现在在你的祭坛上给你的蜡烛和三枚不同的银币充电。

在它们充电的时候，给自己放一次澡。 一旦放好，将你的三枚银币和一撮盐放入水中。 把绿色蜡烛放在浴缸的脚下。现在点燃绿色蜡烛，然后关掉你浴室里的其他灯光。

躺在水中，专注于蜡烛的火焰，并想象你收到这笔钱。 吟唱：“绿龙飞过海洋-为我带来财富和繁荣”

注意：不要清洗，只是浸泡在水中。 一旦蜡烛燃烧了四分之一的长度，就离开浴缸并擦干自己。 拿起蜡烛和你的硬币放在你的祭坛上。 允许蜡烛燃烧下去。

--------------------------------------------------------------------------------

一条说明：如果你没有祭坛或者不知道什么是“给你的硬币充电”，你仍然可以做这个金钱咒语。拿一张漂亮的纸，在背面写上“给我充电！”。翻过来，把你的硬币放在那上面放在书架或窗台上。那样就行了。
